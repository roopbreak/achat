# 무협 서술 스타일 가이드

무협 세계관 스토리 생성 시 참조하는 복식·체형·성적 묘사 용어 레퍼런스.
현대적 표현을 무협 톤으로 치환할 때 이 가이드를 기준으로 삼는다.
**현대 무협 웹소설에서 실제로 통용되는 표현 위주로 정리.** 고전 문헌 전용 한자어는 배제.

---

## 1. 복식

### 1-1. 겉옷

| 용어 | 설명 |
|------|------|
| 포(袍) | 겉옷 총칭. 장포, 흑포, 백포 등 색을 앞에 붙임 |
| 도포 | 선비풍 겉옷. 정파 무인 |
| 경장 | 가벼운 전투·이동 복장 |
| 포의 | 서민의 거친 천옷 |
| 전포 | 전투용 겉옷 |

### 1-2. 여성 복식

| 용어 | 설명 |
|------|------|
| 의상/옷 | 일상 복장. "담색 의상", "흰 옷" 등 색+의상으로 묘사 |
| 경장 | 활동적인 가벼운 복장. 여무인, 호법 등 |
| 궁장 | 궁중풍 화려한 복장. 귀족, 기방 주인 등 |
| 나삼 | 얇고 비치는 옷. 유혹, 침실 장면에서 사용 |
| 잠옷 | 잠자리 복장 |

### 1-3. 속옷

| 용어 | 설명 |
|------|------|
| 속의 | 속옷 총칭. 가장 범용적 |
| 속적삼 | 저고리 안에 받쳐 입는 상의 |
| 가슴가리개 | 가슴을 감싸는 여성 속옷. 가장 직관적인 표현 |

### 1-4. 허리띠·장신구

| 용어 | 설명 |
|------|------|
| 혁대 | 가죽 허리띠. 무인 |
| 요대 | 허리띠 일반 |
| 비녀 | 머리 장신구. 옥비녀, 금비녀 |
| 옥패 | 허리에 차는 옥 장식 |

### 1-5. 탈의 표현

| 표현 | 용례 |
|------|------|
| 옷고름을 풀다 | 가장 흔한 탈의 표현 |
| 매듭을 끄르다 | "등 뒤의 매듭을 끄르자 가슴가리개가 흘러내렸다" |
| 포를 벗기다 | "젖은 포를 벗겨 내렸다" |
| 홑겹 하나만 걸친 | 반쯤 벗겨진 상태 묘사 |
| 나신 | 완전히 벗은 상태 |

**탈의 순서**:
- 남: 겉옷(포/경장) → 속적삼 → 속의
- 여: 겉옷(의상/경장/궁장) → 속적삼 → 가슴가리개/속의
- 여(나삼 차림): 나삼 → 가슴가리개/속의 (속적삼 생략 가능)

## 2. 체형 묘사

### 가슴 크기 (컵 사이즈 대체)

| 현대식 | 무협풍 묘사 |
|--------|------------|
| A컵 | 아직 채 여물지 않은 가슴, 소녀의 풋풋한 가슴 |
| B컵 | 소담한 가슴, 단아한 곡선 |
| C컵 | 단단히 여문 가슴, 옷 위로 드러나는 곡선 |
| D컵 | 풍만한 가슴, 경장을 밀어 올리는 곡선 |
| E컵 | 탐스러운 가슴, 옷고름 사이로 넘칠 듯한 풍만함 |
| F컵 | 의복으로 감추기 어려운 풍만한 가슴, 걸음마다 출렁이는 |

### 체형 표현

| 현대 표현 | 무협풍 |
|-----------|--------|
| 글래머 | 풍만한 몸매, 탐스러운 몸매 |
| 슬림 | 가녀린 허리, 버들 같은 허리 |
| 탄탄한 | 단단하게 다져진 몸 |
| 흰 피부 | 백옥 같은 살결 |
| 갈색 피부 | 밀빛 피부, 햇볕에 그을린 피부 |
| 예쁜 얼굴 | 빼어난 용모, 수려한 이목구비 |

## 3. 성적 묘사

### 3-1. 행위 지칭

| 용어 | 뉘앙스 |
|------|--------|
| 합환 | 가장 범용적. "합환을 치르다" |
| 운우지정 | 대표적 완곡어. "운우지정을 나누다" |
| 정사 | 직접적이지만 무협에서도 자연스럽게 혼용 |

### 3-2. 성기 명칭 (통일 규칙)

| 대상 | 무협 표현 | 비고 |
|------|----------|------|
| 남성 성기 | **자지** | 현대어 통일. 장르 불문 |
| 여성 성기 | **보지** | 현대어 통일. 장르 불문 |

### 3-3. 가슴 묘사

직접적으로 "가슴", "유방"을 그대로 사용. 무협이라고 특별히 다른 단어를 쓸 필요 없음.
- "풍만한 가슴", "탐스러운 유방", "봉긋한 가슴"
- 유두: "유두" 그대로 사용. "꼭지"도 허용
- 음핵: "음핵" 그대로 사용

### 3-4. 탈의·정사 장면 묘사

무협 정사 장면의 핵심은 **옷 벗기는 과정**.

**탈의 묘사 패턴:**
- 옷고름을 풀다 → 의상이 흘러내리다 → 속적삼 사이로 가슴이 드러나다 → 가슴가리개의 매듭을 끄르다
- "떨리는 손으로 옷고름을 풀자 의상이 어깨에서 미끄러졌다"
- "속적삼 하나만 걸친 그녀의 몸이 달빛에 비쳤다"

**복장 스테이터스 예시:**
```
👗 복장: 경장 상의 벗겨짐, 속적삼만 걸친 상태. 요대 느슨히 풀림
👗 복장: 잠옷 한 겹. 옷고름 풀려 가슴골 드러남
👗 복장: 속의만 걸친 채. 가슴가리개 매듭 풀림
```

### 3-5. 체액·신체반응

| 용어 | 비고 |
|------|------|
| 정액 | 그대로 사용 |
| 애액 | 그대로 사용 |
| 홍조 | 흥분으로 피부가 붉어짐 |

## 4. 프롬프트 적용 — 치환 규칙

### 복식·장소

| ❌ 현대 표현 | ✅ 무협 표현 |
|-------------|-------------|
| C컵 | 단단히 여문 가슴, 옷 위로 드러나는 곡선 |
| 속옷 | 속의, 속적삼, 가슴가리개 |
| 단추 풀림 | 옷고름 풀림, 매듭이 끄러짐 |
| 속옷 노출 | 속의 드러남, 가슴가리개만 걸침 |
| 브래지어 | 가슴가리개 |
| 팬티 | 속곳 |
| 지퍼 | 옷고름, 매듭 |
| 여관/호텔 | 객잔 |
| 카페 | 다관, 찻집 |
| 레스토랑 | 주루(酒樓), 주막 |
| 침대 | 평상, 침상 |

### 성적 표현

| ❌ 현대/일반 표현 | ✅ 무협 표현 |
|-----------------|-------------|
| 자지/남성기 | 자지 (현대어 통일) |
| 보지/여성기 | 보지 (현대어 통일) |
| 섹스/성관계 | 합환, 정사 |
| 오르가즘 | 절정 |
| 사정 | 사정 (그대로) |
| 애액 | 애액 (그대로) |
| 침대가 삐걱이다 | 평상이 삐걱이다 |
