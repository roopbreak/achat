## 12. V4 특수 태그 (Advanced NAI V4 Features)

NAI V4에서만 작동하는 관계 및 와일드카드 기능.

### 능동/수동/상호 (source/target/mutual)

멀티캐릭터 이미지에서 누가 누구에게 행동하는지 지정.

```
source#: A가 B에게 행동할 때
target#: B가 A에게 행동 받을 때  
mutual#: A와 B가 서로 행동 (양쪽 캐릭터에 동일 작성)
```

**예시:**
- `source#: A character name, fingering, vaginal, sex`
  → A가 주동으로 행동함
- `target#: B character name, receiving, moaning`
  → B가 수동으로 행동 받음
- `mutual#: kissing, hugging, caressing`
  → 둘 다에 동일하게 적용됨

### 텍스트 삽입 (text#)

이미지에 텍스트나 대사 표시.

```
text#: "character name: dialogue here"
```

### 와일드카드 / 랜덤 선택 (||옵션1|옵션2||)

태그를 작성할 때 여러 옵션 중 랜덤하게 적용.

```
||short hair|long hair||  # 짧은 머리 또는 긴 머리
||blue eyes|green eyes||  # 파란 눈 또는 초록 눈
```

## 13. 프롬프트 작성 팁

### 태그 우선순위
1. **캐릭터 정보** (이름, 외형, 성격)
2. **장면 설정** (배경, 상황, 시간)
3. **핵심 행동** (체위, 행위, 표정)
4. **세부사항** (의상, 재질, 신체 특징)
5. **품질 태그** (화풍, 렌더링 옵션)

### 성행위 장면 기본 구성
```
[캐릭터], [체위], [감정표정], [피부상태], [신체특징], [의상상태]
```

예: `character name, sex from behind, torogao, sweat, large breasts, 
clothing aside`

### 영어 순서의 중요성
- NAI V4는 **앞쪽 태그를 더 강하게 반응**
- 가장 중요한 요소를 앞에 배치
- 쉼표(`,`)로 구분

```
uncensored genitals, explicit genitals, visible vagina, visible penis,

```
