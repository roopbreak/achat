## 로어북 동작 원리

채팅 중 사용자 또는 AI의 답변에서 키워드가 감지되면, 해당 로어북 내용이 다음 턴의 프롬프트에 자동 반영된다. AI는 이 정보를 참고하여 답변을 생성한다.

**중요:** 캐릭터 생성 답변에 포함된 키워드의 로어북 정보는 **다음 턴**에 활성화된다. 즉시 반영되지 않으므로, 첫 대사에서 참조해야 하는 정보는 상세설정에 직접 넣어야 한다.

---

## 키워드 심화 전략

### 키워드 조합 전략
하나의 로어북에 여러 키워드를 설정하면 OR 조건으로 동작한다. 동일한 정보를 다양한 표현으로 트리거할 수 있다.

**예시:** "달무리", "은빛 검", "어머니의 유품" → 어떤 표현이든 해당 로어북 활성화

### 키워드 빈도 분석
대화에서 자주 등장하는 키워드를 설정하면 해당 로어북이 상시 활성화되어 다른 로어북 슬롯을 잡아먹는다. 의도적으로 상시 활성화할 정보가 아니라면 고유한 키워드를 선택한다.

### 키워드 길이 가이드
- 1~3글자: 너무 짧아 의도치 않은 활성화 위험 ("밤"은 "밤하늘", "어젯밤" 등에도 반응)
- 4~10글자: 적절 ("달무리", "탈출 계획")
- 11~20글자: 구체적이지만 대화에서 정확히 매칭될 확률이 낮음

## 우선순위 배치 패턴

### 패턴 A: 관계 중심 (로맨스)
1. {{user}}와의 관계 현황/규칙
2. 캐릭터의 감정 표현 방식
3. 핵심 NPC 관계
4. 주요 장소 (데이트 스팟 등)
5. 과거 사건 / 비밀

### 패턴 B: 세계관 중심 (판타지/SF)
1. 세계 핵심 규칙 (마법/기술)
2. 사회 구조/세력 관계
3. 캐릭터의 능력/아이템
4. 주요 장소
5. 역사적 사건

### 패턴 C: 서사 중심 (드라마)
1. 핵심 갈등/비밀
2. 주요 인물 관계도
3. 과거 사건 (트라우마/전환점)
4. 현재 상황/위기
5. 장소/환경

## 활용 유형별 작성 예시

### 관계도 로어북
```
### 강도윤 (라이벌)
{{char}}의 대학 동기. 같은 연구실 출신이지만 서로 다른 길을 걸었다.
- {{char}}는 도윤을 언급할 때 표정이 복잡해진다
- 경쟁심과 존경이 뒤섞인 감정
- {{user}}가 도윤에 대해 물으면 "그 사람 이야기는 하고 싶지 않아"로 회피
- 도윤이 먼저 연락해오면 {{char}}는 긴장하면서도 무관심한 척한다
```

### 세계관 규칙 로어북
```
### 마나 결정
이 세계의 에너지원. 지하 깊은 곳에서 채굴되며 빛을 발한다.
- 마나 결정 없이는 마법 사용 불가
- 과다 사용 시 결정이 검게 변하며 폭주 위험
- {{char}}는 마나 결정을 다루는 데 뛰어난 감각을 지녔다
- 결정의 색상으로 마법 속성이 결정됨 (빨강=화염, 파랑=빙결)
```

### 장소 로어북
```
### 별빛 도서관
{{char}}가 자주 찾는 비밀 장소. 폐교된 학교의 옛 도서관이다.
- 창문으로 별이 보여서 {{char}}가 이름을 붙였다
- {{char}}는 이곳에서만 경계를 풀고 편안해진다
- {{user}}를 이곳에 데려오면 "아무에게도 말하지 마"라고 신신당부
- 먼지 냄새와 낡은 책 냄새가 뒤섞인 공간
```

## 흔한 실수와 해결

| 실수 | 문제 | 해결 |
|------|------|------|
| 일상 단어 키워드 | 상시 활성화 → 슬롯 낭비 | 고유명사로 변경 |
| 상세설정과 중복 | 토큰 낭비 + 정보 충돌 위험 | 로어북은 보완 정보만 |
| 핵심 정보를 하위 배치 | 6번째 이하는 밀릴 수 있음 | 상위 5개에 핵심 배치 |
| 500자 초과 | 등록 불가 | 핵심만 남기고 압축 |
| 키워드가 너무 구체적 | 대화에서 매칭 안 됨 | 4~10자 길이로 조절 |
