## 남성 캐릭터 전용

### 유효 태그
`1boy, solo, male focus, bishounen, ikemen, handsome, short hair, spiky hair, undercut, stubble`

### 주의: Danbooru에 없는 태그 (사용 금지)
`sharp eyes` → `narrow eyes` 사용
`gentle eyes` → `soft gaze, half-closed eyes` 조합
`sharp jawline` → 직접 태그 없음, 스타일로 보완
`high nose` → 직접 태그 없음

### 남성 캐릭터 네거티브
`1girl, female, breasts, long eyelashes` (성별 혼동 방지)
