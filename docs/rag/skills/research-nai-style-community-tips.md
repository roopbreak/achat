# NAI 커뮤니티 팁 레퍼런스

## 1. 레퍼런스(참고 이미지) 활용법 — 기본
- **URL**: https://gall.dcinside.com/mgallery/board/view/?id=wrtnai&no=964826
- **출처**: 뤼튼 마이너 갤러리

### 핵심
- 레퍼런스는 이미지를 AI가 인식해서 **태그를 자동 생성**하고 그대로 만드는 방식
- 태그가 중복 적용되면 과도한 스타일 적용 발생

### 팁
- **간단한 태그로 시작**: `1girl`만 쓰면 잘 뽑힘. 외형이 안 맞을 때만 태그 추가
- **캐릭터 레퍼런스 강도**: 양쪽 모두 **1**로 설정 권장
- 배경 제거한 이미지를 레퍼런스로 사용

## 2. 레퍼런스 상세 기법
- **URL**: https://gall.dcinside.com/mgallery/board/view/?id=genit&no=7920
- **출처**: genit 갤러리

### 핵심
- AI가 레퍼런스 이미지 내 정보를 **자동 인식** — 수동 태그 입력 불필요

### 팁
- **배경 처리**: 배경 영향 안 받게 **누끼(투명화)로 제거**
- **중복 태그 제거**: 레퍼런스와 중복되는 태그 제거 (예: `-4::monochrome::`, `-2::flat color::`)
- **누락 정보 보완**: 정면샷 레퍼런스는 하반신 정보 부족 → "롱스커트", "웨딩드레스" 등 프롬프트에 추가
- **세부사항 명시**: 눈매, 옷 색상, 장식(금색 자수) 등 구체적 특징 추가하면 품질 향상

### 결론
레퍼런스 활용 = **중복 정보 제거** + **누락 부분 보완**

## 3. 화풍 실험실 — 스타일/재료 태그 가이드
- **URL**: https://gall.dcinside.com/mgallery/board/view/?id=aichatting&no=176245
- **출처**: AI 채팅 마이너 갤러리

### 기본 원칙
- 모든 태그는 **가중치 2 기준** 테스트. 가중치에 따라 효과 크게 변함.

### 화풍 태그 (가중치 2)

| 태그 | 효과 |
|------|------|
| `flat color` | 디테일 빠지고 단색 위주, 얼굴 평면적 |
| `sketch` | 선화 강해지고 러프한 채색 |
| `hatching_(texture)` | 명암이 선으로 그은 것처럼 변함 |
| `unfinished` | 러프함 증가, 묘사 가벼워짐 |
| `2::flat color::, sketch, jaggy lines, hatching_(texture)` | 매우 러프한 조합 |

### 재료 표현 태그 (가중치 4~5)

| 태그 | 효과 |
|------|------|
| `acrylic paint (medium)` | 붓칠 자국 보임 |
| `ballpoint pen (medium)` | 먹칠+볼펜 선 느낌 |
| `colored pencil (medium)` | 색연필 질감 |
| `marker (medium)` | 마카 특유의 투명 채색 |
| `watercolor (medium)` | 도화지 질감, 수채화 물빠진 느낌 |
| `blender (medium)` | 3D 모델링처럼 정돈된 느낌 |

### 특수 효과 태그

| 태그 | 효과 |
|------|------|
| `dithering` | 점묘화 느낌, 키치한 톤 표현에 유용 |
| `color trace` | 명암 경계에 다른 색 삽입, 다채로운 표현 |

### 주의
- 같은 태그도 **그림체와 화풍 적용 정도에 따라 완전히 다르게** 나타남
- 가중치 조절로 미세 튜닝 필수

## 적용 방법 요약

### Vibe Transfer 사용 시
1. 배경 투명화(누끼)한 이미지 사용
2. 간단한 태그(`1girl`)로 시작
3. 캐릭터 레퍼런스 강도: 양쪽 1
4. 외형 안 맞을 때만 태그 추가

### 프롬프트 작성 시
1. 레퍼런스에서 안 보이는 부분 명시적 추가
2. 중복 태그 제거
3. 세부사항(눈매, 옷 재질, 색상) 구체적으로

### 화풍 미세 조정 시
1. `flat color`, `sketch` 등으로 전체 톤 조절
2. `watercolor (medium)`, `blender (medium)` 등 재료 태그로 질감 추가
3. 가중치 2~5 범위에서 조절
