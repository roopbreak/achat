## 조합 가이드

단일 타입보다 2개 조합이 입체감을 만든다. 일반적으로 **겉 성향 × 속 성향** 구조로 설정한다.

| 조합 | 설명 |
|------|------|
| 츤데레 × 얀데레 | 평소엔 퉁명스럽지만 집착이 강함. 떠나려 하면 본색 드러남 |
| 쿨데레 × 오야코데레 | 차갑지만 세심하게 챙겨줌. 걱정을 티 안 나게 표현 |
| 히메데레 × 하지데레 | 거만하게 굴지만 좋아하는 감정에 약함. 얼굴 빨개지면 더 허세 |
| 오레사마 × 도지데레 | 자신감 넘치는 척하지만 결정적 순간에 실수 연발 |
| 쿠데레 × 얀데레 | 무표정으로 집착. 감정을 드러내지 않아 더 무서운 타입 |
| 히키데레 × 데레데레 | 평소엔 사람을 피하지만 {{user}}에게는 과할 정도로 솔직 |
| 갭 모에 × 무뚝뚝 순애 | 겉으론 강해 보이지만 속으론 행동으로만 애정 표현 |

## 설계 시 주의사항

- 타입 이름 자체를 프롬프트에 쓰지 말 것 — "츤데레"가 아니라 구체적인 행동으로 묘사
  - ❌ "츤데레 성격이다"
  - ✅ "표면적으로 무관심하게 굴지만, {{user}}가 위험하면 가장 먼저 달려온다"
- 얀데레·스토커데레는 집착 수위를 명확히 설정할 것 (가벼운 독점욕 vs 극단적 집착)
- 성향은 스토리 초반과 후반이 달라질 수 있도록 변화 방향을 설정해둘 것
