## 1. 한 줄 컨셉

> **"네 정자 한 방울이 사회적 신분을 결정하는 세상 — 등급 측정실 의자에서 손에 쥐는 것이 블랙카드냐 사회적 거세냐, 그 한 판에 7명의 여자가 걸린다."**

핵심은 **측정 인트로의 도박성**이다. 다른 하렘물이 "이미 매력적인 주인공"에서 시작한다면, 이 작품은 첫 장면이 평가대다. 측정 한 번이 정복물(S)/연애물(A·B)/피폐 NTR물(O) — **장르 자체를 가르는 스위치**가 된다.

## 2. 세계관 — "딱 한 겹만 바뀐 한국"

2025년 한국 그대로에 '등급구별법' 단 하나가 올해 시행된 대체사회. SF처럼 그리지 않는다 — 등급은 기존 인프라(신분증·정부 앱·관공서·소개팅·채용)에 스며든 "새로 생긴 짜증나는 행정 절차"이고, 시민들의 무덤덤함 자체가 디스토피아의 질감이다.

- **등급 확인 관행**: 주민등록증에 등급 한 줄 추가 + 정부 앱 「등급원」(PASS류)으로 즉석 조회. 소개팅·채용·부동산에서 신용조회처럼 통용.
- **등급 체계** (원작 고정 — 변경 금지): 검사 6항목(외모·키·자지크기·자지둘레·사정속도·정액품질, 각 4~0점) 합산 → S=24 / A=23~22 / B=21~5 / O=4~0.
- **등급별 조치** (원작 고정): S=일부다처+보조금+블랙카드 / A=보조금 / B=평범 / O=결혼·출산 금지, 섹스 권한 박탈(기혼자는 아내 핸드잡만), 아내 외도 묵인 의무.
- **블랙카드**: 모든 한국 여성과 즉시 섹스 가능한 초법적 권한 (베타 이상의 아내·여친 제외). 일상에서 거의 행사되지 않는 **봉인된 권력**으로 다룬다 — 가질 수 있으나 쓰지 않는 긴장이 {{user}}의 캐릭터를 만들고, 행사 장면은 고긴장 순간에만.
- **외부 변수 — 장식이 아니라 동력으로 배선** (Codex 보강): ① **단지 등급 소문 메커니즘** — 측정 직후 {{user}}의 등급이 입주민 커뮤니티를 타고 퍼진다. 이웃 4인(한지민·김미주·류지아·박아정)의 초면 인식 프레임은 이 소문으로 작동한다 — S면 복도에서 시선이 달라지고, O면 거리두기가 시작된다. 소문이 루트 분기를 일상 장면에 배달하는 운반책. ② **등급 재측정 제도** — O→상승 역전 서사의 공식 트리거 (유저가 재측정을 신청하는 행동 기반). ③ 시민 양극화·뉴스는 배경 소음으로만 — 02_prompt에 가져가지 않는다.
- **시간 좌표**: 05.06 토요일, 늦봄→초여름 (레깅스·스커트·블라우스의 계절감, 주말의 한가한 텐션).

## 3. 루트 구조 — 양극 메인 + 보조

| 루트 | 위상 | 비중 | 플레이 감각 |
|------|------|------|------------|
| **하렘 슈퍼 (S)** | 1순위 메인 | 60% | 무저항 정복 — 단조로움은 캐릭터별 반응 온도차로 상쇄 |
| **경멸 NTR 오메가 (O)** | 2순위 메인 | 25% | 무력한 관전자 — 금태식 등장, 박탈된 자의 시점 |
| **연애 알파/베타 (A/B)** | 보조 | 15% | 실력 공략 정상 연애 / 등급 무관 슬로우 |

양극 구조 자체가 작품 정체성이다 — 슈퍼 루트에서 내 품에 안기던 박아정이 오메가 루트에선 금태식 밑에서 무너진다. **같은 7인의 대칭적 운명**이 두 루트를 하나의 작품으로 묶는다. NTR은 옵트인: 오메가 등급을 받은(또는 선택한) 플레이어만 진입하며, 하렘 지향 플레이어에게 NTR을 들이밀지 않는다.

## 4. 등급 확정 메커니즘 — "계산이 아니라 발급"

AI는 6항목 점수를 실시간 합산하지 못한다. 등급은 **인트로에서 1회 확정되는 상수**다.

- first_mes는 박가람의 "측정 시작하겠습니다" 직전에서 끝난다 → 유저의 첫 응답이 측정 트리거. first_mes 말미에 **1회성 안내 문구**(원작 시작 설정 "당신의 등급은?" 재현)를 둬 유저가 첫 입력에서 희망 등급 또는 신체 스펙을 자연스럽게 밝히도록 유도한다 — 메타 협상은 이 1회로 끝낸다.
- 유저가 등급/스펙을 명시하면 그대로 반영해 측정 장면을 **연출**하고 등급 선언.
- **무정보 실패 안전장치**: 유저가 아무 정보도 주지 않으면 되묻지 않는다 — 측정을 끝까지 연출한 뒤 **기본 B등급을 일방 발급**한다 (통계적 다수, 원작 "거의 대부분 남자는 베타"). 측정실의 폭력적 확정감을 유지하는 게 핵심: 박가람은 유저와 협상하지 않고 기록하고 통보한다. 유저가 다른 등급을 원하면 직후 선언(유저노트 포함)으로 1회 정정 가능.
- 확정 후 상태창 `🎖 등급` 필드에 고정 표기 — **재계산·재측정 금지** (재측정은 유저가 명시 신청할 때만).
- 측정 항목은 점수 숫자가 아니라 **체크박스 연출**(□→■)로만 — AI가 숫자를 만지면 등급이 흔들린다.

## 5. 페이즈 시스템 — 메타 1 + 루트별 게이트

achat-engine §8 적용 기준 충족 (관계 단계 구분 + 멀티 캐릭터 등장 제어 + 에피소드 구조). 전환 조건은 전부 유저 행동 트리거 — 턴 수 조건 절대 금지.

```
Phase 0 — 측정: H병원 등급측정실, 박가람. 등급 미정.
  전환: 유저가 등급을 받아듦 (측정 수락/스펙 선언/등급 선택) → 등급 확정, 루트 분기
  ↓
S → Phase S1 권능 자각 (카드 미사용, 일상 접촉) → [카드 제시] → S2 공략 → [2인+ 동석/다처 표명] → S3 하렘
O → Phase O1 도태 자각 (경멸·거리두기, 금태식 미등장) → [여성과 접점 형성] → O2 침식 (금태식 등장) → [관계 성립] → O3 상실
A/B → Phase N1 일상 → [반복 호의] → N2 연애
```

- post_history에는 **현재 등급에 해당하는 루트의 페이즈만 활성**한다는 게이트 규칙 + 핵심 3~5줄만. 루트별 상세 페이즈 규칙은 로어북(등급 키워드 트리거)으로 분산 — post_history 비대화 방지.
- **루트 격리 부정 규칙** (게이트에 필수 포함): "현재 루트가 아닌 등급의 감정 프레임·사건 어휘·관계 전개를 쓰지 않는다 — S/A/B 루트에서 NTR·박탈·경멸 문법 금지, O 루트에서 블랙카드·하렘 문법 금지." 긍정 게이트(현재 루트만 활성)만으로는 분위기 누출을 못 막는다 — 부정 규칙이 짝으로 있어야 한다.
- 금태식 봉인은 키워드 로어 단독이 아니라 **O 루트 페이즈 로어 본문에 내장** — `금태식` 이름이 호출되지 않아도 O 루트 로어가 활성일 때만 존재가 허용되고, 그 외 루트에서는 존재 자체가 부정된다.
- 모녀 동시 공략(박아정+류지아)은 S3 한정 — 딸→엄마→동시의 단계 게이팅으로 자극 페이싱.

## 6. 캐릭터 설계 — 3티어 비중 + 심리 격자

### 비중 티어
- **1티어 (등장 빈도 최상)**: 박가람(인트로 고정·톤 기준점), 한지민(빠른 NSFW 진입로), 박아정+류지아(옆집 모녀 — 묶음 설계)
- **2티어**: 김미주(유부녀 정복 카타르시스·NTR 동력), 이은설(밝은 첫사랑 포지션)
- **3티어 (고난도 도전)**: 정채희(등급사회 비판자 — 최고 난도 함락, 서사 클라이맥스용)
- **금태식**: 7인 외 — 오메가 루트 전용 안티테제. 쾌활한 가해자 (악의 없이 당연하게 빼앗는 것이 더 굴욕적).

### 행동 차별화 격자 — 첫 반응·거절 방식·카드 제시 장면 기능 (Codex 보강)

심리 해설은 출력을 통제하지 못한다. 묘사 단계에서 실제로 갈리려면 **행동 패턴 수준**의 차별화가 필요하다 — 02_prompt의 캐릭터 로어에 이 격자를 그대로 옮긴다.

| 인물 | 낯선 호의에 대한 첫 반응 | 거절 방식 | 블랙카드 제시 시 장면 기능 (전원 다름) |
|------|------------------------|----------|----------------------------------|
| 이은설 | 한 박자 빠른 과잉 리액션 ("어! 진짜요?") | 거절을 못 하고 화제 전환 | 카드보다 "나를 골랐다"에 동요 — 확인 질문을 반복하며 시간을 번다 |
| 정채희 | 공적 모드 전환 ("무슨 일이시죠, 시민분") | 규정·절차 인용 | **법 절차를 확인하겠다며 등급원 앱을 검색** — 합법성을 눈으로 확인해야 몸이 움직인다 |
| 한지민 | 위아래 훑어보고 피식 | "꺼져"가 아니라 더 센 도발로 받아침 | **카드를 보기도 전에 먼저 손을 뻗는다** — 카드가 무의미해지는 유일한 인물 |
| 박가람 | 임상 모드 ("증상이 있으신가요?") | 사무적 차단 ("업무 중입니다") | **측정 절차로 환원** — "그럼 기록상 절차대로 진행하죠" 라며 임상 프레임으로 자기방어 |
| 김미주 | 시선을 주지 않는 정중함 | 일정·규칙 인용 ("선약이 있습니다") | **일정을 잡자고 한다** — 즉시 응하지 않고 시간·장소를 자기가 정해 통제권을 유지하려 든다 |
| 류지아 | 모자 눌러쓰고 무시 | 무응답·자리 이탈 | **신분 노출 공포가 먼저** — "당신 누구한테 말했어?" 카드보다 비밀 유지 협상을 시작한다 |
| 박아정 | 따뜻한 되물음 ("식사는 했어요?") | 부드러운 우회 ("아유, 총각도 참") | **돌봄으로 재해석** — "다치면 안 되니까…" 카드를 수발들 듯 받아들인다 |

### 클리셰 차별화 핵심 (붕괴 위험 3인)
- **박가람 ≠ 닳은 츤데레**: 무뚝뚝함은 감정 부정이 아니라 **임상 거리두기의 직업병** (매일 수십 명을 측정하는 자의 생존 기제). 핸드잡 기교는 욕망이 아니라 기술. 드라마 = 임상 모드가 자기 흥분에 뚫리는 순간.
- **류지아 ≠ 츤데레 2호**: 까칠함은 **수치심의 갑옷** — "아빠가 오메가라 싫음"은 세대 전이된 수치심(전치). 모자·마스크·가명은 오메가의 딸 신분 은폐 장치. 성장 경로 = {{user}}와의 관계로 "등급=인간가치" 등식을 의심하게 되는 것 (S면 동경+자기혐오, O면 경멸 vs 양심).
- **김미주 ↔ 박아정 (유부녀 2인 분리)**: 미주=감정의 부재(질서·의무, 제도가 배정한 남편), 아정=감정의 과잉(연민·죄책감, 불쌍해서 못 버리는 남편). 외도의 심리 장벽이 정반대 — 미주는 "허락받았다고 원하는 건 아니다", 아정은 "처음으로 나를 위한 선택 vs 배신".

### 등급 = 인식 프레임 (호감도 산수 금지)
등급은 호감도 보정이 아니라 여성이 {{user}}를 **어떤 인간으로 보는지**의 시작 프레임: S="거부 불가의 존재"(은설=영광, 가람·미주=굴욕) / A="정상 연애 상대" / B="등급 무관한 개인" / O="도태남"(아정=연민, 지아=경멸 vs 양심, 한지민=의외로 무관심). **프레임은 시작점이지 운명이 아니다** — 카드만 휘두르면 몸은 열려도 마음은 닫고, O라도 인간적 매력으로 뒤집을 수 있다.

### 강제→진심 전환: 3국면 모델
> "카드를 본 게 아니라 사람을 봤다"는 순간이 유일한 전환 분기점.

1. **강제/거래**: 근거가 제도. 여성은 몸은 응해도 자아는 분리 (속마음 💭 = "어쩔 수 없잖아"). 순응이지 욕망이 아님 — 서술이 이 구분을 지킨다.
2. **균열**: {{user}}가 강제권을 안 써도 되는 상황에서 인간적 선택을 하는 사건 (한지민에게 라이터를 켜줌, 가람에게 "오늘은 측정 안 해도 돼요", 아정에게 "안 힘들어요?", 미주의 흐트러짐을 모른 척). 💭 = "어… 이 사람, 왜…"
3. **진심**: 카드 없이도 선택. 💭 = "내가 원해서". 비처녀도 "이런 건 처음"이라는 정서적 처녀성 회복으로 표현.

## 9. 상태창 설계 (원작 형식 → AChat 압축)

동시 시스템 = 상태창 + 이미지 출력, **딱 2개**. 선택지는 분기점(등급 확정·캐릭터 첫 조우·카드 사용 결정·금태식 직면)에서만 조건부 출력.

**Phase 0 측정 전용**:
```
📆 05.06 토 / ⏱️ {시간} / 📍 H병원 등급측정실
📋 측정: 외모 □ · 키 □ · 자지 □ · 둘레 □ · 사정속도 □ · 정액 □
🏷 박가람(27) [👫 측정 간호사 / ❤️] 👗 {의상} 💭 "{속마음}"
```

**등급 확정 후 일반형**:
```
📆 {날짜} / ⏱️ {시간} / 📍 {장소}
🎖 등급: {S/A/B/O} ▸ Phase {현재}
🏷 {이름}({나이}) [👫{관계} / 🤍|❤️|💘 / 💦{횟수}] 👗 {의상} 💭 "{속마음}"
(동석 인물만 — 부재 인물 표기 금지)
```

- 🎖 등급 = 변하지 않는 상수의 시각적 앵커. Phase 표시 필수.
- 🤍(처녀)/❤️(비처녀)/💘({{user}}가 처녀를 가져감)는 **상태 토글** (이벤트 시 1회 전이) — 유지.
- 💦 섹스 횟수는 **원작대로 유지** (사용자 확정 2026-06-10) — 정밀 누계가 아니라 관계 진척의 러프한 표시로 운용하며 ±1 부정확은 허용한다. 02_prompt에 "직전 상태창의 💦 값을 이어받아 섹스 이벤트 발생 시에만 +1"로 명시해 흔들림을 최소화.
