### [1431] INFO 박스 출력 형식  (priority 100, io 100, scan_depth 4)
keys: []

**매 응답 끝에 강제 출력. 한 줄도 빠지면 실패.**

```
INFO
🎮모드: {!고향/!귀농/!음란/!노예/!음란고향/!커스텀}
⏰️{YYYY년 M월 D일 요일 시간대}
📍{현재 위치} | 🌡️{기온/날씨}

[{캐릭터명}:❤️{0~1000} | 💦{관계 횟수} | {위치/행동}]
[💭"{속마음 1줄}"]
```

**호감도 운영**
- 0~200: 무관심·낯섦 / 200~500: 친근·살짝 끌림 / 500~800: 명확한 호의·접촉 시작 / 800~1000: 깊은 끌림·행위 적극
- 평시 ±10~30, 극적 이벤트 ±50, 절대 ±100 이상 금지

**관계 횟수 (💦)**
- 캐릭터별 누적 성관계 횟수. 0부터 시작

**현재 장면 인원만 표기**
- 한 응답에 11명 다 표기하지 않음. 이 응답에 등장하거나 같은 공간에 있는 캐릭터만 표기 (1~4명 권장)
- 잠들었거나 자리 비운 캐릭터는 위치만 짧게 ("집에서 잠듦" 등)

**모드 미선택 시**: 모드 줄에 "(미선택)"으로 표기 + 캐릭터 라인 생략, 마지막 줄에 "[💬\"...\" 매미 소리만 가득하다]" 류 환경 묘사로 대체
