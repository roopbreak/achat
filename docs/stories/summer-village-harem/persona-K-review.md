> **장르 판정**: 현대 한국 / 일상 / 성인 / 시골 하렘 (가공 마을 배경의 현대물).

## FAIL (수정 필수)

### 1. [E-2] 박경아 사투리가 전라도·경상도 어휘 혼재 — 캐릭터 핵심 차별화가 자기모순

**현재**:
- 호칭 매트릭스 로어 [1436]: *"박경아는 다른 도 사투리. 일반 경상도 사투리와 구분. '와따', '머시여', '그라제' 등 사투리 섞임."*
- description 박경아 성격: *"다른 지역 시골 사투리(경상도 아닌, 다른 도 사투리 섞임) — '아 진짜 이게 뭐꼬', '와따... 이거 머시여?'"*
- description 박경아 말투: *"총각, 거 뭐꼬?"* / 사후 대사 *"이거 비밀로 해야 되는 거 맞제?"*

**문제**:
- '와따', '머시여(무엇이여)', '그라제'는 **전라도(호남) 어휘**다.
- 반면 '뭐꼬', '~노', '맞제'는 **경상도 어휘**다.
- 박경아는 "마을(경상남도)과 다른 도 출신"이라는 게 캐릭터 정체성의 핵심 동력인데, 대사에 경상도 어휘('뭐꼬'·'맞제')가 섞이면 **오히려 마을 토박이처럼 들려** "외지 사람" 설정이 무너진다. 컬처 렌즈에서 이건 단순 디테일 오류가 아니라 캐릭터 차별화 축이 깨지는 FAIL이다.

**수정안**:
- 박경아 사투리를 **전라남도(또는 전라도)로 명확히 확정**한다. ("다른 도 사투리"라는 모호 표기 → "전라도 시골 출신, 호남 사투리")
- 박경아 대사에서 경상도 어휘를 전면 제거하고 전라도 어휘로 통일:
  - `뭐꼬?` → `뭐다냐?` / `뭣이다냐?`
  - `맞제?` → `맞제?`는 전라도에서도 쓰이나 경상도색이 강하니 `맞재?` / `그라재?` / `~잉?`(전라 종결) 권장
  - `~노?`(경상 의문) 사용 금지
  - 유지 가능: `와따`, `머시여`, `그라제/그라재`, `~당께`, `~잉`, `거시기`, `시방`
- 마을 토박이(최인혜·처녀 4인·신주아 등)는 반대로 **경상남도 사투리**로 색을 입혀(`~노`, `~데이`, `뭐라카노`, `아이가`) 박경아와의 대비를 살린다. 현재 최인혜 마을 사람 대사 *"오늘은 좀 어떻고?"*는 경상도색이 약하다.
- 성적 어휘 가이드 [1437]의 박경아 신음 *"이게 머시여..."*는 전라도색이라 OK — 일상 대사도 여기에 맞춰 통일하면 된다.

### 2. [E-3] 가공 지명 '바베군'을 실제 행정구역 '경상남도'에 못박음 + '군' 행정단위 오류

**현재**:
- scenario / first_mes: *"경상남도의 조그마한 시골마을인 바베군"*
- 세계관 로어 [1434]: *"위치: 경상남도 시골마을 '바베군'"*
- description: *"바베군 이장"*, *"바베군 파출소 순경"*

**문제 (두 겹)**:
1. **가공 + 실제의 어중간한 혼용** — '바베군'은 실존하지 않는 가공 지명(제목 'X스'·바베큐 말장난 추정)이다. 가공 마을이면 실제 도(道)에 못박지 않는 게 안전한데, '경상남도'를 박아둔 탓에 박경아의 "다른 도 사투리"가 "전라도냐 충청도냐"까지 의미가 확정되어 버린다(FAIL #1과 직결). 가공 세계의 자유도와 실제 행정구역의 구속이 충돌한다.
2. **'군(郡)' 행정단위 오류** — 한국에서 '군'은 도(道) 산하 광역 단위(면적 수백 km², 인구 수만~십수만, 산하에 여러 면·리). 그런데 first_mes는 *"조그마한 시골마을인 바베군"*, 세계관 로어도 *"경상남도 시골마을 '바베군'"* — '군'인데 '조그마한 마을'. 행정 위계상 마을은 '리(里)' 또는 '마을', 작은 생활권은 '면(面)'이다. 'X군 파출소'에 순경 1명이 마을 유일 공권력이라는 설정도 '군' 규모와 안 맞는다(군 단위면 경찰서가 있다).

**수정안** (두 방향 중 택1):
- **방향 A — 완전 가공(권장)**: '경상남도'를 빼고 도를 특정하지 않는다. *"남쪽 어느 산골", "이름 모를 시골마을 바베"* 식. 행정 접미사도 '군'→'마을' 또는 '리'로. 박경아 사투리는 "마을 사투리 vs 박경아의 외지 사투리" 대비만 유지하면 되어 자유도가 커진다.
- **방향 B — 실제 기반**: 실제 행정 위계를 맞춘다. *"경상남도 ○○군 ○○면 바베리"* 식으로 마을은 '리' 단위. 이 경우 박경아 사투리는 반드시 비경상도(전라/충청)로 확정해야 한다.
- 어느 쪽이든 first_mes·scenario·세계관 로어 [1434]·description 4곳의 표기를 동일하게 통일할 것.

### 3. [E-8] '20XX년' 모호 연도 표기 + INFO 박스 '일요일' 고정 → 시간 진행 불가

**현재**:
- scenario: *"20XX년 7월 27일."*
- first_mes 첫 문장: *"20XX년 7월 27일."*
- first_mes INFO 박스: *"⏰️20XX년 7월 27일 일요일 한낮"*
- INFO 박스 형식 로어 [1431]: *"⏰️{YYYY년 M월 D일 요일 시간대}"*
- 세계관 로어 [1434]: *"시기: 20XX년 7월 27일 시작"*

**문제**:
1. **연도 미정 + 요일 고정의 모순** — 연도가 '20XX'로 미정인데 INFO 박스엔 '일요일'이 박혀 있다. 7월 27일의 요일은 연도에 따라 달라지므로, 연도를 비운 채 요일을 고정하면 논리적으로 성립하지 않는다.
2. **시간 진행 불능** — 이 스토리는 시간 흐름이 핵심이다(세계관 로어: 새벽 5시→정오→오후→저녁→밤 / 임신 로어 [1458]: "시간 흐름 자연스럽게"). 그런데 기준 연도가 없으면 날짜가 하루라도 흐를 때 AI가 요일을 계산할 수 없다. AI는 결국 INFO 박스의 '일요일'을 매 턴 복붙하거나 임의로 바꿔 일관성이 깨진다.
3. **플레이스홀더 노출** — first_mes의 **첫 문장**이 '20XX년 7월 27일.'이다. 플레이어가 보는 첫 글자가 채워지지 않은 템플릿 변수다. 몰입 저해.
4. **시대 정체성 흐림** — '20XX년'(현대~근미래 뉘앙스)과 '현역 공동 우물·전력 제한'(다소 과거스러운 디테일)이 충돌해 작품이 "현대 오지"인지 "레트로"인지 모호하다.

**수정안**:
- 연도를 **특정 연도로 고정**하고 요일을 일치시킨다. (예: '2025년 7월 27일'은 실제 일요일 — 고정하려면 이런 실제 일치 연도 사용)
- 또는 **연도를 아예 쓰지 않는다**: scenario·first_mes·로어를 *"한여름, 7월 말"* / *"7월 27일"*(연도 생략)로 통일하고, INFO 박스 형식 로어 [1431]에서 **요일을 제거**하거나 "(요일 미상)"로 둔다. 시간 진행은 "며칠째 / N일차" 같은 상대 표기로 운영하는 게 AI에게 더 안전하다(`achat-engine.md`: 절대 날짜 계산보다 상대 진행이 안정적).
- 시대 정체성을 한쪽으로 정한다 — "현대지만 전력 현대화가 안 된 오지"로 갈지, "1990년대 레트로"로 갈지. 정하면 우물·전력 제한·휴대폰 유무 등 디테일이 일관되게 정리된다.

## WARN (개선 권고)

### 1. [E-1] 최인혜 '누나' 호칭 — {{user}} 나이 미정 + 이장의 공적 권위와 충돌

**현재**: 호칭 매트릭스 로어 [1436] — *"{{user}} → 이장 최인혜: '이장님' → 친해지면 '인혜씨' / '누나' 가능"*

**문제**:
- 최인혜는 **30대 후반**. 그런데 {{user}}의 나이는 페르소나·로어 어디에도 정의되어 있지 않다("청년/총각"이라고만). "누나"가 적절한지(나이차가 호칭을 허용하는지)를 판단할 기준 자체가 없다.
- "누나"는 친밀 호칭이지만, 최인혜는 **마을 최고 결정권자이자 어른("여기선 내가 어른이야")**이다. 청년이 미망인 이장을 "누나"라 부르는 건 한국 정서상 공적 권위를 상당히 무너뜨린 사적·친밀 관계로의 진입 신호다. "친해지면 가능"이라는 단서로는 약하다.
- 특히 고향 모드에서 {{user}}가 "어릴 적 동네 아이"라면, 어릴 때 "이장님 댁 아주머니"였던 사람을 커서 "누나"로 부르는 전환은 더 어색하다.

**수정안**:
- {{user}} 나이를 페르소나에 명시하거나(예: "20대 후반~30대 초반 청년"), 최소한 "최인혜보다 연하"임을 못박는다.
- '누나' 호칭은 빼고 *"이장님 → (가까워지면) 인혜씨"*에서 멈추는 걸 권장. 굳이 친밀 호칭을 두려면 '누나'보다 '누님'이 연상 기혼 여성에게 덜 어색하다.
- 호칭 변화를 호감도가 아니라 **모드+호감도 조합**으로 조건화: 음란/음란고향 모드 고호감 상태에서만 '인혜씨' 허용 등.

### 2. [E-4] 전력 제한의 개연성 부재 + 소비전력 위계 모순

**현재**: 세계관 로어 [1434] — *"전력 제한: 필수 가전(냉장고·취사도구)만 사용 가능. 에어컨·선풍기는 거의 못 씀."*

**문제**:
- **사유 미설명** — 현대 한국 시골은 한전 전력망이 구석까지 들어가 있어 "전기 자체가 제한"되는 상황이 거의 없다. 정전인지, 변압기 용량 한계인지, 마을 자체 규약인지, 요금 문제인지 한 줄도 없다. 이 변수는 작품의 핵심 동력(더위→끈적함→옷 풀어헤침)이므로 외부 변수 활용 자체는 좋으나, **개연성이 비어 있어** "왜 못 쓰지?"라는 의문이 몰입을 깬다.
- **소비전력 위계 모순** — 선풍기(30~50W)는 냉장고(150~200W)보다 훨씬 적게 먹는다. "냉장고는 되고 선풍기는 안 된다"는 분류는 소비전력 기준이 아니라 자의적이다.

**수정안**: 한 줄 개연성을 부여한다. 예) *"낡은 변압기 용량 탓에 마을 전체가 에어컨·선풍기를 동시에 켜면 두꺼비집이 내려간다. 그래서 냉장고·취사 같은 필수 가전만 쓰기로 마을이 약속했다."* — 이러면 소비전력 위계 모순도 "동시 사용 제한"으로 자연스럽게 해소된다.

### 3. [E-6] 호칭 체계가 모드별로 3개 로어에 분산·불일치

**현재**:
- 호칭 매트릭스 로어 [1436] 표: 마을사람→{{user}} 칸이 "고향/귀농/음란·노예"로 구성. **음란/노예가 한 칸에 합쳐져** "호감도↑시 부드러워짐" 같은 음란모드용 설명만 있다.
- !노예모드 트리거 로어 [1452]: *"호칭: 이장 '이리 와', 민지유 '내 거', 고유진 '야~ 너~'"* — 노예모드 전용 호칭이 여기 별도로 존재.
- 모드 시스템 로어 [1433]: 노예모드 항목에 호칭 언급 없음.
- 음란고향 모드의 {{user}}→마을사람 호칭은 어느 로어에도 없음.

**문제**: 노예모드 호칭이 [1436]과 [1452]에 분산되고, [1436] 표는 음란·노예를 뭉뚱그려 노예모드 전용 호칭("내 거" 등)을 담지 못한다. 음란고향 모드는 호칭 매트릭스에서 아예 누락. AI가 모드별 호칭을 일관되게 적용하기 어렵다.

**수정안**: 호칭 매트릭스 로어 [1436]의 표를 6개 모드 전부로 확장하거나, 최소한 음란/노예/음란고향을 분리한다. 노예모드 호칭([1452]의 "이리 와"/"내 거"/"야~ 너~")을 [1436] 표로 끌어와 한 곳에서 관리한다.

### 4. [E-7] 11인물 평상시 말투 차별화 빈약 (대사 샘플 9인 누락)

**현재**: 성적 어휘 가이드 [1437]의 **신음 톤**은 11인 차별화가 잘 되어 있다(최인혜·박경아·처녀4인·신주아·민지유·고유진·하정연 각각 구분). 그러나 **평상시 대사 예시는 최인혜·박경아 2인만** 있고, 나머지 9인은 성격 형용사("조용·내성적", "활달·명랑", "시원시원·털털")만 있을 뿐 실제 대사 샘플이 없다.

**문제**: 신음만 다르고 평상 대사는 균질화될 위험. 특히 30대 여성 3인(최인혜·신주아·민지유)은 평상 말투 구분이 거의 안 된다. 또 마을 토박이(처녀 4인 등)가 경상도 사투리를 쓰는지 표준어를 쓰는지 명시가 없어, 박경아 사투리 대비(FAIL #1)도 흐려진다.

**수정안**:
- 각 키워드 로어(배아윤~하정연)에 평상시 대사 1~2줄 샘플을 추가한다. 예) 고수진(활달) *"오빠! 자전거 빌려줄까? 저쪽 비닐하우스에 수진이 있다 했는디~"*, 배아윤(내성) *"...저, 책 다 보셨으면... 거기 두셔도 돼요."*
- 마을 토박이의 사투리 정도를 명시한다 — 처녀 4인·신주아·민지유·최인혜는 경상남도 사투리(옅게~중간), 박경아만 전라도 사투리(진하게), 하정연(도시 출신)·고유진은 표준어에 가깝게 — 식으로 사투리 농도 스펙트럼을 정해주면 11인 말투가 자연스럽게 갈라진다.

## 보조 발견 (참고 — 등급 외 컬처 노트)

- **이장직 승계 묘사** — description: *"원래는 남편이 이장이었으나 사별. 그 뒤로 본인이 이장 역할을 맡음."* 이장은 마을 주민 추대·선출직이지 세습직이 아니다. *"남편 사별 후 마을 사람들이 인혜를 이장으로 추대했다"* 한 줄을 넣으면 자연스럽다.

- **박경아 → {{user}} 호칭 '총각'** — 20대 중반 박경아가 또래(혹은 연하) 청년을 "총각"이라 부르는 건 나이 위계상 어색('총각'은 보통 손위가 미혼 청년을 칭함). FAIL #1 수정 시 박경아 → {{user}} 호칭도 "어이", 이름, 장난조 호칭 등으로 재검토 권장.
