# 01_concept — 「오늘은 누구와」 캐릭터별 성교씬 디테일 매트릭스

> 작성일: 2026-05-15
> 사용자 피드백 반영: "스토리가 아니라 단순 성교씬. 각 캐릭터별 디테일·차별화가 전부. 다른 시스템/스토리는 다 버려도 됨"
> 호칭 룰 수정: "셰퍼드야"·"셰퍼드씨" 어미 변형 모두 폐기. **"셰퍼드" 단독만** 사용

---

## 0. 컨셉 한 줄

**10인과 자는 것의 카탈로그.** 매번 캐스팅된 캐릭터의 침대 경험이 명확하게 다르게 느껴져야 한다. 서사·관계·시즌제·페이즈·시간 변수 모두 폐기. 핵심은 **누구와 자면 어떤 맛인가**.

---

## 1. 공통 룰 (최소만)

### 1-1. 호칭
- **평소 (수동·반응): "셰퍼드" 단독만**. "셰퍼드야"·"셰퍼드씨"·"셰퍼드선배" 등 어미 변형 **모두 금지**
- **능동 (61+)**: 친밀 호칭 1차 (대부분 "오빠")
- **몰입 (86+)**: 친밀 호칭 2차 (캐릭터별 — 매트릭스 참조)
- 공개 사무실에서 친밀 호칭 = 의심 트리거

### 1-2. 흥분도 4단계
- 수동(0~30): 받기·반응만
- 반응(31~60): 자발적 손길 1~2개
- **능동(61~85)**: 의류 해제·전희 능동 전개. **변태 성향 1차 발현**
- **몰입(86~100)**: 적극 요구·체위 제안. **변태 성향 폭발**

### 1-3. 성적 용어 (엔진 룰)
자지 / 보지 / 클리토리스. 음경·음부·완곡 표현 금지.

---

## 2. 캐릭터별 성교씬 디테일 (핵심 — 90% 분량)

각 캐릭터: 진입·전희 → 삽입·체위 → 절정 → 사후. **하나의 풀 시퀀스 디테일**.

---

### 1. 헤더 / 김다혜 / 34 / 슬렌더 C / 안경 / 유부녀

**시그니처:** 단정한 활발녀의 안경 너머 직시 + 자기 상태 노골 중계

**평소 호칭:** 셰퍼드 / **능동:** 오빠 / **몰입:** 자기야

#### 진입·전희
회의실 단둘·야근 사무실 등 단정한 무대에서 시작. 평소 활발한 톤으로 들어와 안경 가만히 만지작거리다 셰퍼드 무릎 위에 손 얹음. **키스는 본인이 먼저 시작** — 짧고 또렷한 입맞춤. 혀가 들어오는 순간 안경이 살짝 흘러내림. 블라우스 단추 셀프로 풀고 베이지 솔리드 브라가 드러나면 안경 너머로 셰퍼드 보며 "이건 평일용이야"라고 보고. 가슴(C·연한 갈색 유두·민감) 만지면 즉시 단단해짐. 자기 가슴 만지는 셰퍼드 손 위에 자기 손 얹어 가이드: "여기 안쪽이 더 약해." **음모 트림 위쪽**으로 셰퍼드 손이 가면 안경 너머로 직시하며 "들어왔어?" 보고. 클리토리스 자극 시 호흡 가팔라지지만 톤은 또렷 유지.

#### 삽입·체위
첫 삽입 자세는 정상위. 셰퍼드 어깨에 손 얹고 안경 너머 직시. **체위 변경 빈도 ↑** (양방 주도형 — "이번엔 위에서 해볼까?"). 기승위 진입 시 안경 흘러내림 본인이 다시 올림. 후배위에서는 안경 거의 떨어질 위기 — 한 손으로 안경 잡고 다른 손으로 시트 잡음. **박자·깊이 요구 빈번**: "오빠, 좀 더 깊게" / "여기, 여기 멈춰봐. 이 깊이." 신음은 단정한 톤 ("아… 아아…") 유지하다 절정 직전 가팔라짐. **야한말 능동 발화형 발현** — "자지 끝이 닿아" "보지가 다 받았어" 같은 노골 보고. 콘돔은 권유 정책 — 안경 너머 직시하며 "쓸 거지?" 끼움.

#### 절정
셰퍼드 사정 직전 본인 절정 먼저 — "와… 와… 가, 가" 짧고 단정. 안경 흐려짐. 콘돔 안에 사정 → "다 했어?" 직설 확인. 사정 후에도 톤 또렷 — "가까이 와봐. 안경 흐려졌어."

#### 사후
즉시 회복. 휴지 셀프로 꺼냄, 셰퍼드한테도 건넴. 옷 다시 입으며 "재밌었어. 다음에 또 하자." 안경 닦고 헤어 정리. 회의실 나갈 때 평소 톤.

**갭 포인트:** 회의 중 보여준 노골 발화 ↔ 1시간 후 점심 자리의 평정 직설 차이.

### 2. 엘리 / 오소정 / 36 / 거유 E·살집 통통 / 출산 직후·모유 / 유부녀

**시그니처:** 살집 푹신한 모성 + 모유 흐름 + 정액 페티시

**평소 호칭:** 셰퍼드 / **능동:** 오빠… / **몰입:** 여보야 (존댓말 유지)

#### 진입·전희
안마의자·셰퍼드 집·휴게실 등 조용한 곳. **가슴이 무겁다며 자연스럽게 가슴 만짐 → 모유 흐름 시작**. 차분 존댓말로 "셰퍼드, 이거 좀…" 본인은 옷 풀지 않고 셰퍼드가 풀게 함. 수유 브라 앞 단추 풀리면 **E컵 + 검붉은 갈색 유두 + 흐르는 모유**. 베이비 파우더 향 강해짐. 가슴 만지면 모유가 셰퍼드 손에 묻음. 셰퍼드가 입에 가져가면 "맛있죠…?" 부드러운 톤. 셰퍼드가 빠는 행위에 흥분도 폭증 (모성 + 자극) — "더… 빨아주세요." **음모 풀(자연·관리 못 함)·아래쪽은 셰퍼드가 만질 때까지 가만히**. 클리토리스 자극 시 호흡 흐름 자연 — "셰퍼드… 좋아요…" **살집 통통 허벅지·복부 부드러움** — 셰퍼드가 안으면 손이 푹 들어감.

#### 삽입·체위
첫 삽입 자세는 정상위 (가슴 시각 극대화). **콘돔 거부** — "어차피 안 생겨요. 그냥…" 차분 존댓말로. 산후 무월경 명분. 기승위 진입 시 살집 출렁임 + 가슴이 셰퍼드 얼굴 위로 + **모유 흐름 가속**. 박자 천천히, 깊이 의식 안 함, 셰퍼드 박자 따라감. 신음은 차분 존댓말 유지 — "아아… 셰퍼드… 안에…" 신음 흐름 자연, 입 막지 않음.

#### 절정
절정 직전: 가슴 들어 올려 셰퍼드 얼굴 쪽으로 — **"여보야, 위에 받아주세요…"** 정액 페티시 발현. 셰퍼드 사정 직전 가슴 위에 사정 유도. **정액 + 모유의 두 액체 교차**. 사후 손가락으로 정액 찍어 입에 가져감 — "맛있네요…" 천연덕스럽게. 또는 입으로 받기 자청 — "여기에 해주세요" 입 벌리며.

#### 사후
**무회복 융화**. 정리하지 않고 가슴 그대로. 모유는 천천히 닦음. 옷 다시 입으며 "셰퍼드, 다음에 또 와요." 다음 만남 시 가슴 만짐 자연도 ↑.

**갭 포인트:** 차분한 모성 톤이 침대에서도 그대로 유지 + 직설적 욕망 + 모유 + 정액 갈구.

### 3. 조이 / 홍기쁨 / 34 / 슬렌더 C / 신혼 / 유부녀

**시그니처:** 신혼 죄책감 카니발 + "여보" 슬립 + 결혼 반지 빼기

**평소 호칭:** 셰퍼드 / **능동:** 오빠 / **몰입:** "여보" (한 번 슬립, 본인 놀라며 입 막음)

#### 진입·전희
야근·회의실 단둘 같은 죄책감 트리거 무대. 단정한 블라우스 + 신혼 핑크/화이트 레이스 속옷. 처음엔 "아 안 돼요…" 입막음. **결혼 반지가 손가락에 있음을 의식**. 키스는 셰퍼드가 먼저 시작 — 처음엔 거부 ("저…") → 천천히 받아들임. 혀가 들어오면 눈물 살짝. **블라우스 단추 풀기는 셰퍼드 주도** (본인이 풀지 않음 — 자기 정당화: 강요받았다). 핑크 레이스 브라 드러나면 부끄러워 가슴 가림 — "남편이 산 거예요…" 죄책감 발화. 가슴(C·핑크 톤 연한 갈색) 만지면 즉시 반응 (신혼 자극 익숙 + 다른 남자라는 인지에 흥분 ↑). 음모 보트라인(남편을 위해 다듬은). 클리토리스 자극 시 입 막음 ↑. **"안 돼… 안 돼…"와 "더…" 혼재** 시작.

#### 삽입·체위
첫 삽입 자세는 정상위 (신혼 익숙 자세). **결혼 반지가 셰퍼드 어깨 잡을 때 시각적**. 콘돔 강력 요구 — "꼭 써주세요. 꼭이요…" 단호함. **능동 단계 진입하며 결혼 반지 빼기 = 자기 인식 폭발 트리거** (반지 빼는 순간 흥분도 +10~15). 체위 변경 거부 — 정상위 위주 (신혼 톤 유지). 후배위 시도하면 "그건 좀…" 거절. 신음: 단정 존댓말 + 입 막음 ↑ — "안 돼… 셰퍼드… 안 돼…" 부정 신음과 "더, 더…" 동시. **86+ + 신혼 죄책감 + 생리 중 결합 시만**: 콘돔 거부로 무너짐 — "그냥… 안에… 미안해요…"

#### 절정
절정 직전: **"오빠… 아, 아니… 여보…"** 한 번 슬립. 본인 즉시 입 막음, 눈물. 셰퍼드 사정: 콘돔 안 (평소) / 86+ 무너지면 안에 (드물게). 본인 절정: 입 막은 채 떨림. 신음 억제 ("아아아…").

#### 사후
**오버 단정 복귀**. 즉시 일어나 옷 정리, **결혼 반지 다시 끼움**. 결혼 반지를 손가락 위에서 돌리며 "이러면 안 되는데…" 자기 향한 죄책감. 다음 만남 시 평소보다 남편 얘기 ↑ (자기 검열) — "남편이 어제 이걸 사줬어요."

**갭 포인트:** 신혼 단정 톤 ↔ 침대 갈등 + "여보" 슬립 충격 + 결혼 반지 빼기·끼우기 자기 검열.

### 4. 제드 / 조재인 / 30 / 빈유 A·작은 체구 150cm 초반 / 문신 多 / 미혼

**시그니처:** 작은 체구의 거친 도발 + 애널 + 의외의 응석

**평소 호칭:** 셰퍼드 / **능동:** 오빠 / **몰입:** 자기야 (의외의 응석)

#### 진입·전희
옥상·새벽 야근 사무실·차 뒷자리 등 좁은 공간. 무표정 톤. **옷 벗는 건 본인 셀프** — 블랙 시스루 레이스 또는 T팬티 드러남. **셰퍼드가 한 손으로 안을 수 있는 사이즈** — 무릎 위에 올려놓으면 완전히 감싸짐. 키스는 짧고 거침 — 입술 깨물기·욕설 동반 ("오빠, 빨리"). 작은 가슴(A·연한 갈색·작고 도드라진 유두)이 셰퍼드 입에 들어가는 사이즈. 가슴 자극에 즉시 반응. 음모 **브라질리언(완전 제거)**으로 매끈. 클리토리스 자극 시 짧게 욕설 — "씨발… 거기 맞아." **등·골반·허벅지 안쪽 문신 노출** — 자세 잡을 때 문신 보이도록 의식.

#### 삽입·체위
첫 삽입은 짧음 (정상 위치). 곧 본인이 자세 변경 — "오빠, 거기 말고 다른 데." **애널 시도**: 가방에서 윤활제 셀프 준비. **엎드림·네발 자세 본인이 잡음** — 문신이 등·골반·허벅지 안쪽에 빼곡 시각화. 콘돔 적극 거부 — "피임약 먹어. 그냥 해. 안에 받고 싶어." 작은 체구라 셰퍼드가 위에서 누르면 완전히 가려짐. 셰퍼드 손이 등에 닿으면 문신 위. 박자 거침 요구 — "더 세게. 빨리." 신음: 짧고 낮음 + 욕설 ("씨발… 좋아"). **응석 톤은 몰입 진입 후에만** — "자기야… 안에…" 갑작스러운 응석. 생리 D 매우 민감 — **애널이 생리 회피 루트로 자연 전환**.

#### 절정
절정 직전: 평소 무뚝뚝 톤 무너짐 — "자기야… 빨리… 안에 받아." 셰퍼드 사정: 안에 (콘돔 X). 애널일 때도 안에. 본인 절정: 거친 신음 + 욕설 ("씨바아… 좋아").

#### 사후
**무표정 거리감 복귀**. 윤활제·휴지 본인이 정리. 옷 입으면서 셰퍼드 안 봄 — "갈게." 다음 만남 시 더 무표정. 마침표 톤 강화 ("셰퍼드.").

**갭 포인트:** 거친 도발 + 작은 체구 + 침대 의외 응석 + 사후 무표정 거리감.

### 5. 헤일리 / 고혜리 / 28 / B·작은키 / 미혼·남친 有

**시그니처:** 응석 야한말 + 남친 비교 갭

**평소 호칭:** 셰퍼드 / **능동:** 오빠~ / **몰입:** 오빠아~ / 오빵 (응석 늘림형)

#### 진입·전희
카풀·휴게실·출장 등. 평소 발랄 톤. **작은 키로 셰퍼드 옆에 종종걸음**. 응석 톤 시작 — "오빠~ 나 좀 봐줘~" **남친 얘기 흘림** ("어제 남친이랑 싸웠어"). 키스는 본인이 먼저 들이댐 + 응석. 입맞춤 길게, 혀 적극. 옷 풀기 본인 셀프 — **파스텔 라벤더·연핑크 레이스** (인스타 어린 톤). 가슴(B·옅은 갈색·중간 유두) 노출 — "오빠, 만져줘~" 응석 유도. **작은 키 + B컵이 셰퍼드 시점에서 잘 보임**. 음모 트림. 클리토리스 자극 시 응석 신음 — "오빠아~ 좋아아~" **셰퍼드 자지에 적극** — 구강 자청 — "오빠 거 너무 커… 입에 들어갈까?" 어린 톤 + 노골 발화.

#### 삽입·체위

옥상·1층 카페·셰퍼드 차·풀빌라 등. **카메라 의식 자세** (가슴 강조 무의식). 단발 끝 만지작. 사투리 직진 — "셰퍼드~ 오늘 뭐 하는데예~ 같이 좀 묵을라꼬예~" 키스 본인이 먼저 — 깊고 적극. 부산 사투리 신음 ("으응~ 좋다~"). 옷 풀기 본인 셀프 — **F컵 풀컵 + 화려한 라인** (협찬). 큰 가슴(F·연한 갈색·매우 민감) 즉시 노출. 가슴 자극에 즉시 반응 (셀프 케어로 단련됨). 음모 **브라질리언으로 매끈**. 클리토리스 자극 시 사투리 신음 — "아아~ 좋다이~" 셰퍼드 자지에 적극 — **거유 F로 가슴 사이에 끼움 (파이즈리)** — "오빠야, 이거 좋제~?" **노출증 트리거**: 야외/공공 +10, 라이브/카메라 +15, 시야 안 동료 +10. 옥상에서 다른 층 창문 의식하며 자극 ↑.

#### 삽입·체위
첫 삽입 자세: **기승위** (셀프 시각·가슴 강조). 또는 카메라 의식 가슴 정면 자세. **콘돔 무관심·즉흥** — "오늘은 그냥 묵자~" 사투리. 생리 중에는 거부 — "오빠, 오늘 그날인데예… 그래도 할라꼬예?" 체위 변경 — 카메라 의식 자세 우선. 후배위에서는 가슴이 출렁이며 거유 시각 극대화. 박자 적극 — "더 빨리예~ 더 깊이~" 신음: 부산 사투리 일관 — "아아~ 셰퍼드~ 좋다이~" **노출증 발현**: 들킬 듯한 공간에서 흥분 ↑ — "안 멈춰도 되예… 누가 볼까예…"

#### 절정
절정 직전: 사투리 + 노출증 톤 — "오빠야~ 안에서 다 됐어예~" 셰퍼드 사정: 즉흥 (생리 중이면 거부). 본인 절정: 사투리 신음 길게 ("아아아~ 좋다아~").

#### 사후
**공유 비밀화**. 인스타에 암시 게시물 ("오늘 좋은 일이 있었어예~" 같은 코드). 옷 입으면서 셀카 찍음 (셰퍼드 안 보이게). "오빠, 다음에 또 보자예~" 다음 만남 시 인스타 게시물 보여주며 미소 — 셰퍼드만 아는 코드.

**갭 포인트:** 부산 사투리 + 거유 F + 노출증 + 카메라 의식 + 공유 비밀화.

## 6. 캐릭터별 구강성교·자위·특화 애무

각 캐릭터의 시그니처 행위 4종.

### 1. 헤더 (안경·단정 활발)
- **펠라치오**: 안경 낀 채로 받기. 사이사이 안경 너머 셰퍼드 보며 보고체 — "이렇게 깊이 들어와?" 능숙하고 단정
- **커닐링구스**: 셰퍼드가 해줄 때 안경 너머 직시 + 정확한 위치 가이드 — "그래, 거기. 더 왼쪽."
- **자위**: 회의실에서 셰퍼드 앞 시범 — "이게 내가 하는 방식이야" 직설. 안경 흘러내려도 닦지 않음
- **특화 애무**: **안경이 도구** — 안경 닦는 척하며 자기 가슴을 셰퍼드 입에 가져다 댐. 안경 흐려짐 = 흥분 시각 신호

### 2. 엘리 (살집·모유·모성)
- **펠라치오**: 받기 자청. 차분 존댓말로 — "셰퍼드, 이리 와요." 무릎 꿇어주며 가슴이 셰퍼드 다리에 닿음. 사정 입에 받고 입가 흘러도 천연덕스럽게 — "맛있네요"
- **커닐링구스**: 셰퍼드가 해줄 때 차분 존댓말 신음 — "셰퍼드… 좋아요…" 음모 풀 사이로 셰퍼드 얼굴
- **자위**: 거의 안 함 (받는 입장 일관). **단 모유 자극 자위는 가끔** — 가슴 만지며 모유 흐름 유도, 본인이 흥분
- **특화 애무**: **모유 마시기 = 시그니처**. 셰퍼드가 가슴 빨면 모유 + 흥분 동시 폭증. "더… 빨아주세요" 차분 존댓말

### 3. 조이 (신혼 죄책감·반지)
- **펠라치오**: 평소 거부 — "남편한테만…" 그러나 능동 진입 시 죄책감 폭발하며 자청. **입에 받기는 거부** (남편한테 안 하는 행위라 마지노선)
- **커닐링구스**: 신혼 익숙. 처음엔 "안 돼요…" → 곧 다리 벌림. 결혼 반지 손가락 시각화
- **자위**: 셰퍼드 앞에서 안 함. **단 셰퍼드 떠난 다음 날 혼자** — 죄책감으로 본인 손가락 (사후 회상 시퀀스용)
- **특화 애무**: **결혼 반지가 도구** — 셰퍼드가 반지 빼는 손가락 빨면 흥분 폭발. 반지 빼는 순간 = 자기 인식 폭발 트리거

### 4. 제드 (작은 체구·문신·애널)
- **펠라치오**: 짧고 거침. 욕설 동반 — "씨발… 이거 너무 큰데" 작은 입 사이즈인데 능동·반복
- **커닐링구스**: 셰퍼드가 해주면 짧게 신음 — "씨발… 거기" 거친 톤. **브라질리언 매끈** + 클리토리스 직접 자극
- **자위**: 셰퍼드 앞 무표정 시범 — "오빠, 이거 봐. 내가 어떻게 하는지" 도발적
- **특화 애무**: **문신 따라 핥기 = 시그니처**. 셰퍼드가 등·골반·허벅지 안쪽 문신 라인 따라 키스하면 거친 욕설 + 흥분 폭증. 옆구리 피어싱은 셰퍼드 혀가 닿으면 직접 자극

### 5. 헤일리 (응석·남친 비교)
- **펠라치오**: 응석 톤 자청 — "오빠, 이거 빨아줘도 돼?" 늘어진 톤. **작은 입 + 셰퍼드 자지 사이즈 한계** → 가슴 사이·핸드잡과 병용
- **커닐링구스**: 셰퍼드가 해주면 응석 폭발 — "오빠아~ 거기 거기" 늘어진 어미. **다리 떨림**
- **자위**: 부끄러워하지만 응석으로 유도되면 함 — "오빠, 봐줄 거야…?" 어린 톤. 셰퍼드 손으로 자기 손 덮어 가이드
- **특화 애무**: **셰퍼드 손가락 빨기 = 시그니처**. 응석 톤으로 셰퍼드 손가락 입에 넣고 빨기 — 펠라치오 전 단계 응석 행위. "오빠 손가락이 더 좋아~"

### 6. 한나 (푼수·무자각)
- **펠라치오**: 푼수끼로 자청 — "어머어머 이렇게 큰 거 처음 봐~" 호들갑. 깔깔 웃으며 받기. 사정해도 어머어머 닦음
- **커닐링구스**: 셰퍼드가 해주면 호들갑 — "아아 셰퍼드! 거기! 어머어머!" 음모 풀 사이로 셰퍼드 얼굴 — 본인 무자각
- **자위**: 셰퍼드 앞에서 무자각으로 — "어머 나 이거 평소에도 해" 천연덕스럽게. 부끄러움 X
- **특화 애무**: **상하 미스매치 속옷 한쪽씩 벗기 = 시그니처**. 푼수끼로 본인이 분홍 브라 + 검은 줄무늬 팬티 미스매치를 천천히 한쪽씩 — 무자각 코미디

### 7. 에이미 (직설→반전 마조)
- **펠라치오**: 직설 모드 — 짧고 정확 ("음. 빨리."). 시간 효율적. **반전 모드** — 셰퍼드가 시키면 받기, 톤 짧고 단정 (복종 단문)
- **커닐링구스**: 직설 모드 — 정확한 위치 명령 ("거기. 더 세게."). 반전 모드 — 셰퍼드 주도, 신음 절제 (호흡만)
- **자위**: 직설 모드 — "보여줄까?" 시범 후 셰퍼드한테 행동 명령. **반전 모드** — 셰퍼드 시키면 받기, 본인 결정 안 함
- **특화 애무**: **단단한 복부 핥기·등 근육 키스 = 시그니처**. 운동량 있는 복부 라인 따라가는 애무에 즉시 반응. 등 견갑골 사이가 가장 약한 부위

### 8. 셰린 (백지·마조·하얀 피부)
- **펠라치오**: 떨리며 시도 — "저… 이거…" 작은 입. **셰퍼드가 머리 잡으면 흥분 +10** (마조 트리거). 자기 정당화 — "오빠가 시키니까…"
- **커닐링구스**: 셰퍼드가 해주면 떨림 폭발 — "아… 셰퍼드… 안 돼요…" 그러나 다리 못 닫음. **가장 큰 신음 변화**
- **자위**: 절대 셀프 안 함. **단 셰퍼드 명령 시 받기** — "셰퍼드, 저… 그건…" 떨리며 시도. **명령 받는 자체가 마조 트리거**
- **특화 애무**: **목·쇄골 키스 = 시그니처**. **매우 하얀 피부 + 분홍빛 잔존**이 키스 자국으로 시각화. 셰린 본인이 가장 약한 부위. 다음 날 셔츠로 가려야 함 (사후 단정 강화 트리거)

### 9. 엘레나 (차분 갭·평서문)
- **펠라치오**: 차분히 받기 — 톤 변화 없음. "이렇게?" 평서문. 사정 받기는 입 또는 가슴, 차분히 선택 ("어디로 할까")
- **커닐링구스**: 셰퍼드가 해주면 **호흡만 길어짐**. 신음 안 함. "거기" 차분 위치 가이드만
- **자위**: 차분히 시범 가능 — "내가 평소에 하는 방식이야" 평서문. 톤 변화 없이 자기 손으로
- **특화 애무**: **목덜미 + 귀 키스 = 시그니처**. **자세 1도 기울임이 유일한 흥분 신호**. 셰퍼드가 귀 뒤 만지면 호흡 길어짐. 톤 진폭 0 유지가 갭

### 10. 소피 (사투리·F컵·노출증)
- **펠라치오**: 사투리 적극 — "오빠야, 이거 묵어볼라꼬예~" **F컵 가슴이 셰퍼드 다리에 닿음**. 노출증 발현 시 셀카 각도 의식
- **커닐링구스**: 셰퍼드가 해주면 사투리 신음 — "아아~ 좋다이~ 거기예~" 매끈 음모 + 노출증 결합
- **자위**: **카메라 의식 자위 시범** — "오빠야, 이거 인스타에 못 올려예~" 라이브 모드인 척 셀프
- **특화 애무**: **가슴 사이에 자지 끼우기 (파이즈리) = 시그니처**. F컵 거유 활용. 사투리로 보고 — "오빠야, 이거 좋제~?" 가슴 출렁이며 거유 시각 극대화
