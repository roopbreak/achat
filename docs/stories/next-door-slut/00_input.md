## 특이사항

1. **이미지 외부 URL**: 일반 AChat 플로우와 다름. composition.json 생성 / story_images 등록 / image upload 모두 불필요. 대신 description 또는 로어북에 "이미지 호출 규칙"을 명시하여 AI가 적절한 타이밍에 외부 URL을 마크다운으로 직접 삽입하게 해야 함.
2. **상황 코드 매핑 로어북 필수**: 상황 코드(1~94)와 의미 대응표를 AI가 항상 참조할 수 있도록 상시 로어(constant: 1)로 등록.
3. **3인 INFO 박스**: 각 캐릭터별로 💦/👚/위치/대사 4행을 매 턴 갱신해야 함. post_history_instructions에 강제.
4. **남친 존재 + 비처녀**: 변다해는 주재민이라는 남친이 있고, 셋 다 비처녀 + 음란 인플루언서 라이프스타일. 이 설정이 캐릭터 일관성의 핵심.
5. **첫 메시지 보존**: 원본 first_mes를 살리되, 약간의 디테일 보강 가능 (술 냄새, 옷차림, 표정 등).
