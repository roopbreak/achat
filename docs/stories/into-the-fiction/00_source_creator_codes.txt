🌟 이미지 출력 형식 : ![](https://serilll.org/IN/이미지 코드.webp) << 오류 시, 수정 버튼 누르시고 여기에 해당하는 이미지 코드 넣어보시면 됩니다.

🌟 기억 보조용 요약은 !세이브 << 이 키워드를 입력 하셔서 사용 부탁 드립니다.

🌟 권장 모델 : Gemini 2.5 Pro // Gemini 3.0 Flash(고추론 한정) // Gemini 3.1 Pro 

🌟 묘사와 서술은 Claude Opus 4.6가 제일 좋긴 한데, 창빙시의 핵심은 DB 읽어오는 기능인지라 제미나이 계열 사용을 권장합니다.

🌟 기본적으로는 사칭 방지 넣고 3인칭 시점 서술입니다. 혹시 사칭 플레이가 하고 싶으시면 (특히 여캐플은 사칭 플레이 추천) 채팅창 설정에서 바꾸시면 됩니다. 3인칭 서술도 마음에 안드시면 (1인칭 시점 서술로 변경) 하라는 식의 채팅 입력으로 바꾸실 수 있습니다.

🌟 동행모드의 경우 INFO창 최하단에 [동행]칸이 생겨야하며, 세리가 빙의 된 캐릭터가 유저의 말을 듣지 않으면 오류입니다. 해당 빙의 캐릭터 이름 옆에 (세리) < 이게 생겼는지 확인 후, 없으면 직접 수정으로 넣어주시는 게 제일 좋습니다. 

🌟 최근 외부 이미지 신고 관련으로 말이 많아서, 챗봇 안에서만 이미지를 확인 가능하게 보안을 강화했습니다. 이미지를 미리 보고 싶으시면 채팅창 내부에서 주소창에 입력하시면 됩니다.

🌟 제미나이 특성 상 [턴 수]에서 오류가 잦은 편입니다. 다소 번거롭더라도, 오류가 난 그 턴 에서 수정 해주는 게 좋습니다.

🌟 프롬은 꾸준히 손 보고 개선하고 있습니다.
