# 00_input — 너만 아니었다면 (재작성 입력)

- **장르**: 현대/대학/성인 (순애·혐관·구원)
- **구성**: 싱글 캐릭터 (유서린)
- **원작**: babechat "너만 아니었다면"(작가 하진) — 컨셉 기반 충실 포트
- **이미지**: 외부 소스 확보됨(jiyn.org 107장). **NAI 생성 불필요**.
- **핵심**: 완벽주의 강박(어머니 그늘) + 여주인공 연기(계산) ↔ 진심 경계 붕괴 + 구원 순애
- **시스템**: 작가 정의 호감도 7단계 + 경험치30, 상태창 양식(🎭 💋/💢/❤️‍🩹/❤️‍🔥), 구원 시퀀스
- **안전**: 성인 여대생(20) 합의물. 어머니 실물 등장 금지, 강박 미화 금지.
