# PLAN: NSFW 톤 천박·노골 강화

> 작성일: 2026-05-20 | 최종 갱신: 2026-05-20 (Codex 리뷰 반영)
> 대상 파일: `lib/context-builder.mjs` (NARRATION_RULES)

## 1. 목표

성적 장면 톤이 평이하다는 사용자 피드백 → 천박하고 노골적인 톤으로 강화.

### 사용자 요구 (인터뷰 확정)
- **적용 영역**: 캐릭터 대사 + 3인칭 서술·묘사 + 신음·의성어 + 심리·자가 보고 + 이야기 흐름·전개
- **비속어 수위**: 음란어 강하게, 욕설(씨발·좆같은 등)은 절제
- **대사 수위**: 강하게
- **"작품 전체 톤"의 의미**: 일상 대화 천박화가 아니라 이야기 흐름·전개를 노골적으로. 일상 장면 톤은 변경 없음.

## 2. 설계 방향 (Codex 리뷰 반영)

별도 "톤 강도" 섹션 신설 → **각 Tier 항목에 천박·노골 예시를 직접 녹이기** (중복 지시로 모델 집중도 분산 방지). NSFW 오버라이드에는 짧은 원칙 1단락만.

## 3. 수정안

### 수정안 A — NSFW 오버라이드에 "톤 강도" 원칙 (짧게)

```
### 톤 강도 — 천박·노골
**적용 트리거**: 키스·애무·전희·삽입·절정이 명시된 현재 장면, 또는 그 직전 단계의 연속 장면.
가벼운 플러팅·순수 일상 장면은 제외.

- 트리거 장면에서는 평이·점잖은 서술을 금지하고 천박·노골적 밀도를 우선한다.
  구체 적용은 Tier 1/2/3의 강화 예시를 따른다.
- **단계 점프 방지**: "노골적·적극적 전개"는 현재 단계 내부의 밀도 강화만 의미한다.
  단계 상승은 Tier 1 단계 진행 규칙(n→n+1)을 그대로 따른다.
- **분량 유지**: 대사 수위 강화가 분량 보충 우선순위 ①~④(클로즈업·신체반응·자가보고·
  체액분포)를 대체하지 않는다. 짧은 직설 대사로만 분량을 채우지 말 것.
- **안전장치**: 천박 톤은 합의된 성적 장면에만 적용. 거절·주저·동결 반응에는 적용 금지.
  "수치심+욕망 동시"는 캐릭터가 자발적 욕망을 느끼며 부끄러워하는 경우에 한정하며,
  거부 의사를 흥분으로 묘사하지 않는다. 스토리 directive(합의 안전장치·미성년 보호)가
  본 톤 규칙보다 우선.
- **욕설 절제**: 음란어(직설 성적 표현)는 강하게, 거친 욕설(씨발·좆같은 등)은
  캐릭터 성격이 거칠 때만 절제해서 사용.
```
