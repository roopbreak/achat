기존 53행 직후에 별도 규정 추가:
```
- **💭 스테이터스 규정**: 자체 스테이터스가 있는 카드든 기본 형식이든, 💭은 반드시 캐릭터 1인칭 독백(따옴표 권장, 1~2문장). 외부 행동·복장·위치 묘사를 💭에 넣지 않는다. "심장 빨라진다"·"신경 쓰여"·"좋아 보인다" 류 평이한 신체·감정 보고 한 줄로만 채우지 않는다. 가능하면 겉(대사·표정)과 갭 있는 내적 욕망·모순·자기합리화 한 토막을 노출한다. 본문에서 방금 말한 대사를 그대로 복사하는 것도 금지(💬 슬롯이 있다면 거기로).
```

```
[NPC캐릭터명] 👗 (현재 복장 상태) | 💭 (1인칭 속마음 독백. 캐릭터 말투로. 예: "이 남자... 뭐지?", "심장 왜 이래")

→ 신규:
```
[NPC캐릭터명] 👗 (현재 복장 상태) | 💭 (1인칭 속마음 독백 1~2문장. 캐릭터 말투로. 겉모습·대사와 갭 있는 본심·욕망·자기합리화·모순. 예: "거절해야 하는데... 손에서 향수 냄새가 나잖아. 핑계 만들어줘.", "야만인의 눈치고는 맑구나. 아니, 무슨 상관이란 말이냐.", "이러면 안 되는데. 안 되는데도 이러고 있잖아 나는.")
