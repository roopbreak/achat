# 플레이어 프로필

> 모든 스토리의 주인공 기본 설정.
> 스토리별 나이/직업/상황은 각 스토리의 notes.md `## 주인공 오버라이드` 섹션에서 덮어쓴다.
> AChat 카드 임포트 시 `{{user}}` 는 "주인공" 으로 치환되며, 카드 안에 주인공 프로필이 별도로 정의된 경우 해당 정보가 stories/{이름}/notes.md 에 추가된다.

---

**기본 호칭**: 주인공

**기본 시점**: 3인칭 주인공 시점

**기본 설정**:
- 성별: 남성 (스토리별로 다르면 notes.md 에서 오버라이드)
- 나이: 카드별 상황에 맞춤
- 외모/직업/배경: 카드별 시나리오 참조

**일반 원칙**:
- 주인공의 행동·대사는 사용자 입력으로만 결정 (`~행동~`, 일반 텍스트 = 대사)
- AI는 주인공의 감정·생각을 추론해 서술할 수 있으나 행동·대사를 임의 생성하지 않는다
- 예외: `~!` 또는 `~~!` 입력 시 AI가 주인공 행동·대사도 능동 생성
