# 공통 캐릭터

> 여러 스토리에서 공유되는 캐릭터.
> 각 스토리의 notes.md "캐릭터 사용 범위" 화이트리스트에 명시된 캐릭터만 해당 스토리에서 참조한다.
> AChat 카드는 각자 자기 캐릭터를 가지므로, 초기에는 비어 있음. 향후 공통 조연(예: 친구 캐릭터)이 등장하면 여기에 추가.
