# AChat 카드 — 작성 필수 입력 (필요 최소)

> **이 문서는 "무엇을 반드시 적는가"만 정한다.** 작성 절차·안전·연속성·검수는 `card-rewrite-guide.md` 참조.
> 원칙: 컨셉·역학은 카드가 못박고(빠지면 AI가 평균값으로 죽임), 세부 묘사는 AI 자율에 맡긴다. 억지로 칸 채우지 않는다.

## A. 작품 컨셉 (1줄, 필수)
이 스토리가 무엇인가 — 한 줄. (예: "여성 강세 군인 가문의 아내·가족이 사위를 공유하며 능동적으로 주도하는 하렘")

## B. 캐릭터별 필수 (메인·서브 공통)

각 캐릭터에 아래 4개만 반드시. 그 외는 옵션(가이드 참조).

1. **★핵심 컨셉 (갭·포지션·역학)** — 이 캐릭터의 확고한 정체성. 특히:
   - **갭/이중구조** (가장 강한 컨셉 형태): 겉↔속 반전. 예: 겉 청순 우등생 ↔ 단둘이면 음란 / 겉 음란해 보이는데 ↔ 사실 처녀 / 도도 ↔ 침대선 붕괴.
   - **{{user}} 포지션 + 성적 역학 방향**: 누가 주도? 여성상위/남성주도? 무엇을 하는 관계?
   - 예: "라일리 = 남편을 **능동적으로 올라타 주도**하는 여성상위 군인 아내" / "유부녀 = 죄책감 속 빠짐(수동)".
   - ⚠️ "어떻게 무너지나"는 수동형에만. 능동·주도형이면 "어떻게 공략·주도하나". 한쪽 프레임 강제 금지.
2. **말투** — 식별되는 발화 예시 1~2개.
3. **외모·신체 + 성적 톤** — 외모(머리색·눈색·체형) 핵심 + **처녀/비처녀·성경험 수준**(서술 톤 결정: 처녀=통증·서툼·긴장 / 경험 많음=능숙·여유·주도) + 역학에 맞는 신음·대사 샘플 1개. + **컨셉·서사를 드러내는 고유 신체 디테일이 있으면** 명시(예: 무모 보지=관리/노출 / 함몰 유두=특이체질·반전 / 출산한 갈색 큰 유륜·큰 유두=유부녀·엄마). ⚠️ "가슴 큼" 류 평범·무의미한 신체 채우기 금지 — *컨셉을 말해주는 디테일만.*
   - **외모·신체가 카드에 없으면** → 해당 캐릭터의 **등록 이미지를 비전으로 분석해 추출**(머리·눈·체형, NSFW 이미지면 무모/유륜/유두 등까지). 절차는 `card-rewrite-guide.md`.
   - NSFW가 약하거나 모호함이 매력인 캐릭터는 성적 칸을 **"첫 반응·경계·거리감"으로 대체 가능**. 억지 성적 각인 금지.
4. **(멀티만) §C 관계 운영 메모.**

> 컨셉·성향·신체 디테일 발상이 막히면 수집 자료를 **참고**(끼워맞추기 아닌 발상 팔레트): `sexual-dialogue-pool`(성향·갭 유형), `sexual-description-guide`(음모 상태·유륜·유두 등 신체), `character-success-formula`(갭 설계).

## C. 멀티 캐릭터면 추가 — 관계 운영 메모 (3~5줄, 단일 블록)
항목 나열 말고 **하나의 메모로 녹여 쓴다**: 누가 누구를 뭐라 부르고, 누가 누구 앞에서 무엇을 **모르며**(비밀·정보 비대칭), 질투·동맹·견제가 어떻게 도는가. (인물 소개보다 이 관계 축이 멀티의 핵심.)

## D. first_mes
컨셉을 **첫 턴 표면(대사·행동)으로 시연** + 다음 행동 유도. 멀티면 등장 인물 이름 박기.
