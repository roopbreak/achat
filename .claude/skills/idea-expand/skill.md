# Idea Expand — 컨셉 기반 아이디어 확장

사용자가 제공한 대략적인 컨셉/설정을 받아 5가지 서로 다른 방향의 상세 아이디어로 확장한다.

## 실행 절차

### 1. 사용자 입력 분석

사용자가 제공한 컨셉에서 아래 요소를 추출한다. 명시되지 않은 요소는 비워둔다.

- **세계관/배경**: 시대, 장소, 세계 설정
- **캐릭터 특성**: 성격, 외모, 직업, 종족
- **관계 구도**: 유저와의 관계
- **성향/톤**: 분위기, 장르
- **핵심 키워드**: 사용자가 강조한 요소

### 2. 기존 스토리 참고 (선택)

AChat DB에 등록된 기존 스토리와 차별화하기 위해 `GET /api/stories`로 현재 목록을 확인한다.

### 3. 5가지 아이디어 확장

사용자 컨셉의 **핵심은 유지**하면서, 비어있는 요소를 채우고 서로 다른 방향으로 확장한다.

**확장 원칙:**
- 사용자가 명시한 요소는 **절대 변경하지 않는다**
- 각 아이디어는 같은 뿌리에서 **다른 가지**로 뻗어나간다
- 확장 축: 세계관 변형 / 성향 변형 / 관계 구도 변형 / 톤 변형 / 시나리오 변형
- AChat 엔진의 특성(로어북, 이미지 시스템, 메모리)을 활용할 수 있는 방향 우선

**출력 형식:**

```
## 컨셉 분석

**입력된 컨셉:** {사용자 입력 요약}
**고정 요소:** {변경하지 않을 요소}
**확장 가능 요소:** {비어있거나 변형 가능한 요소}

---

### 1. {한 줄 컨셉} — {확장 방향}
- **세계관**: {배경}
- **캐릭터**: {이름 후보} / {성격 한 줄} / {외모 한 줄}
- **관계 구도**: {유저와의 관계}
- **핵심 갈등/매력**: {스토리를 끌고 갈 핵심 요소}
- **AChat 활용**: {로어북/이미지/메모리 중 특히 활용할 시스템}
- **차별화**: {기존 스토리 대비 새로운 점}
- **예상 첫 대사**: "{짧은 예시}"

---

### 2. {한 줄 컨셉} — {확장 방향}
(동일 형식)

... (5개까지)

---

마음에 드는 아이디어가 있나요?
- 번호로 선택하면 바로 캐릭터 제작(`create-story`)을 시작합니다
- 여러 개를 합치거나, 수정 방향을 말씀해주셔도 됩니다
- "N번 더 파줘"로 특정 아이디어를 더 구체화할 수 있습니다
```

### 4. 피드백 반영

| 유저 반응 | 처리 |
|-----------|------|
| 번호 선택 | `create-story` 스킬로 연계 |
| "N번이랑 M번 합쳐줘" | 두 아이디어를 결합한 새 컨셉 제시 |
| "N번에서 성격만 바꿔줘" | 해당 요소만 수정한 업데이트 버전 제시 |
| "N번 더 파줘" | 선택한 아이디어를 하위 변형 3개로 추가 확장 |
| "다시 확장해줘" | 다른 확장 축으로 새 5개 제안 |
| 추가 설정 제공 | 새 설정을 고정 요소에 추가하고 나머지 재확장 |

## 확장 축 매트릭스

5개 아이디어는 아래 축에서 각각 다른 방향을 선택하여 다양성을 확보한다:

| 축 | 변형 예시 |
|----|----------|
| **세계관 변형** | 같은 캐릭터를 현대/판타지/SF/무협 등 다른 배경에 |
| **성향 변형** | 같은 설정에서 순애/집착/갭모에/쿨데레 등 성격 변형 |
| **관계 변형** | 같은 캐릭터와 소꿉친구/직장/적대→연애 등 관계 변형 |
| **톤 변형** | 같은 설정으로 달달/자극적/피폐/개그 등 분위기 변형 |
| **시나리오 변형** | 같은 캐릭터에 다른 핵심 갈등/이벤트 부여 |

사용자 입력에 이미 확정된 축이 있으면 해당 축은 고정하고 나머지 축에서 변형한다.

## AChat 활용 포인트 제안

각 아이디어마다 AChat 엔진 기능을 어떻게 활용할 수 있는지 제안한다:

| 기능 | 활용 예 |
|------|---------|
| **키워드 로어북** | NPC별 정보, 장소별 분위기, 이벤트 트리거 |
| **상시 로어북** | 세계관 규칙, 서술 톤 일관성 |
| **HypaMemory** | 과거 대화 기억 활용, 관계 발전 추적 |
| **이미지 시스템** | 장면별 이미지 매칭, 감정 표현 |
| **서술 명령어** | `~행동~`, `~~~` 장면 전환 활용하는 시나리오 |
| **자동 요약** | 긴 서사에서 맥락 유지 |
