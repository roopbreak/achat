# /st-rollback

직전 교환 쌍(사용자 입력 + AI 응답)을 무효화한다.

ARGUMENTS: `$ARGUMENTS` (선택: 무효화할 턴 수, 기본 1)

## 실행

### 1. 활성 스토리 확인

```bash
cat .playdata/active_story.txt
```

없으면 "활성 스토리가 없습니다." 안내 후 중단.

### 2. 슬롯 경로 결정

`.playdata/{스토리명}/active_slot.md` 존재 시 → `{슬롯경로} = .playdata/{스토리명}/saves/{슬롯명}/`
없으면 → `{슬롯경로} = .playdata/{스토리명}/`

### 3. 컨텍스트 내 무효화

- 직전 N개(기본 1) 교환 쌍을 채팅 컨텍스트에서 제거됨으로 표시
- 카운트 (`[교환: N/15(저장)]`) 도 N개 차감
- 다음 턴부터 이 교환은 컨텍스트에서 *없는 것처럼* 취급

### 4. 세션 파일에서 제거 (백그라운드)

`{슬롯경로}/sessions/session_NNN.md` 의 마지막 N개 교환 블록을 제거.

```bash
# 백그라운드로 세션 파일 마지막 블록 제거
```

### 5. 이전 상태 한 줄 요약 출력

```
이전 N개 교환을 롤백했습니다.
현재 상황: {롤백 후 마지막 상황 한 줄}

다시 이어서 진행할까요?
```

이후 사용자의 새로운 입력을 기다린다.

## 주의

- 이미 15교환 자동 요약이 발생한 후의 턴은 롤백 시 `story_so_far.md` 정합성 주의 (필요 시 사용자에게 확인 후 요약도 보정)
- 롤백된 턴의 영구 흔적은 sessions/ 에서 제거되지만, 이미 만들어진 summary 는 그대로 둔다 (사용자 명시 요청 시만 보정)
