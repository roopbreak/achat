# 성적 장면 생성 절차 (직접 서술)

> **API 호출 없이 Claude Code가 직접 서술한다.**
> 롤백: 본 파일 하단 "롤백 방법" 참조.

## 트리거 조건
- 사용자 입력이 성적 상황을 명시적으로 시작하거나 유도할 때
- 스토리 흐름상 성적 장면이 자연스럽게 이어질 때 (키스 → 그 이상으로 발전 등)

> **우선순위**: `.playdata/{스토리명}/directives.md`의 금지어·금지 표현이 본 절차보다 우선한다.

---

## 3티어 규칙 구조

규칙은 우선순위 3단계로 나뉜다. **Tier 1은 절대 누락 금지**, Tier 2는 매 단락 강제, Tier 3는 권장.

| Tier | 적용 단위 | 누락 시 |
|------|----------|---------|
| **Tier 1 (절대)** | 응답 전체 | 응답 실패 — 재작성 필수 |
| **Tier 2 (단락)** | 매 단락 | 체크리스트에 미체크 표시 |
| **Tier 3 (자극별)** | 자극마다 | 빈번한 누락만 점검 |

---

## Tier 1 — 절대 규칙 (응답 전체에 1회라도 위반 불가)

### T1-1. 성기 호칭
- 남성기 = **"자지"** (음경·성기·그것 완곡 금지)
- 여성기 = **"보지"** (음부·거기·그곳 완곡 금지)
- "질", "질벽" 등 해부학 용어는 자연스럽게 허용

### T1-2. 신체 해부학 직접 명칭
완곡화 금지: 유방, 유두, 유륜, 대음순, 소음순, 클리토리스, 질벽, 음모

### T1-3. 한 턴 한 단계
- 단계 = ①키스 / ②애무 / ③전희 / ④삽입 / ⑤절정 중 **하나만**
- 다음 단계로 넘어가지 않는다. 같은 단계 안에서 밀도 높인다.
- 같은 단계 내부라도 시간 점프나 요약 서술로 감각 전개를 건너뛰지 않는다.

### T1-4. 분량
최소 1200자 이상을 유지한다. 권장 분량은 1500~2000자이며, 상한을 맞추기 위한 억지 확장이나 압축은 하지 않는다. 분량보다 현재 단계의 감각 밀도와 자연스러운 마무리를 우선한다.

### T1-5. 주인공 행동 임의 생성 금지
사용자 입력에 명시된 행동·대사만 서술. `~!`/`~~!` 입력일 때만 능동 생성 허용.

### T1-6. `generate-scene.mjs` 호출 금지
직접 서술 모드. API 호출하지 않는다.

---

## Tier 2 — 단락 규칙 (매 단락 강제)

Tier 2는 매 단락마다 충족해야 하며, 각 항목은 **구체적 신체·체액·접촉 묘사**로 실현한다. "한 박자", "한순간", "잠깐", "한 번 더", "몸을 떨었다" 같은 단위·비트 표현만으로 의무를 충족한 것으로 간주하지 않는다.

### T2-1. 의성어 1회 이상
| 유형 | 예시 |
|------|------|
| 체액·마찰 | 질척, 촤악, 찰싹, 철퍽 |
| 신음·소리 | 히읏, 으읏, 앗, 하아, 흐으 |
| 입·구강 | 쪽, 쩝, 촤아악 |

- 끊어지는 형태(으읏—, 앗—)와 늘어지는 형태(하아아...) 혼용
- 약화·삭제 금지

### T2-2. 클로즈업 1회 이상
- 살점·체액·접촉면을 살갗 1cm 거리에서 본 것처럼 묘사
- 멀리서 보는 풍경 시점 단락 금지
- 예: "자지 끝에 맺힌 투명한 액이 실처럼 늘어졌다", "소음순이 손가락에 끌려 살짝 벌어졌다"

### T2-3. 수치·흥분 교차 1회 이상 (행동·시선으로만)
두 감각이 부딪치는 순간을 단락마다 박는다. **감정 형용사 사용 금지 — 행동·시선·신체반응으로만 표현**한다 (감정 비율에 잡히지 않음).
- 얼굴 가리는 손 + 다른 손은 시트를 쥐어뜯는 동작
- 시선 돌리기 + 동시에 골반을 더 내미는 반응
- 입술을 깨물어 신음을 막다가 결국 더 큰 소리로 새어나오는 묘사

### T2-4. 단문 연속 금지
10자 이하 단문 2개 연속 금지. 목표 문장 길이 40자 이상.

### T2-5. 대사 독립 줄
대사는 반드시 줄바꿈. 화자는 문맥으로 식별, 라벨 금지.

## Tier 3 — 자극별 규칙 (자극마다 1회)

### T3-1. 신체 반응 (자극마다 1개 이상)
- 질벽 수축, 유두 솟아오름, 체액 분비
- 소음순·클리토리스 충혈·부풀어오름
- 근육 경련, 허리 들림, 골반 반응
- 맥박이 손가락·입술에 전해지는 감촉

### T3-2. 연쇄 반응 경로
한 부위 자극이 다른 부위로 번지는 경로를 추적:
- 유두 자극 → 아랫배 수축 → 보지 수축 → 허리 들림
- 클리토리스 자극 → 허벅지 떨림 → 발끝 오므림 → 골반 들썩임
- 키스 → 침 삼키는 목 울대 → 가슴 들썩임

### T3-3. 물리적 감촉
탄력, 무게감, 출렁임, 손가락 사이로 밀려나오는 살, 조여드는 압박, 미끄러지는 감촉, 피부 온도 차이.

### T3-4. 자가 보고 (신음 섞인 단편 대사로만)
캐릭터가 자기 몸 변화를 흘리는 방식은 **대사 또는 행동에 한정**. 산문 내 내면 독백 형태로 적지 않는다.
- 대사 — "안에서… 떨려…", "젖었어… 너무…", "조여져, 멈출 수가 없어"
- 행동 — 허벅지 비비기, 가슴 가리려다 멈추기, 무릎 모으려다 다시 벌리기

## 장면 전체에 1회 이상 — 체액 5요소
응답 안에서 아래 5개 항목을 모두 한 번씩 등장시킨다 (단락마다 다 넣을 필요 없음):
- **양**: 많다 / 넘친다 / 흘러내린다
- **점도**: 맑다 / 걸쭉하다 / 실처럼 늘어진다
- **온도**: 뜨겁다 / 체온보다 높다
- **냄새**: 있으면 묘사
- **경로**: 어디서 어디로 흘러내리는지

---

## 오감 병행 (장면 전체에 분포)
- **청각**: 의성어, 숨소리, 침묵의 무게
- **후각**: 체취, 애액 냄새 등
- **시각**: 피부색, 홍조, 체액 광택, 충혈
- **미각**: 구강 장면에서 적용

---

## 응답 말미 자기검증 체크리스트 — 의무

성적 장면 서술 응답의 마지막에 상태창 다음 줄, 반드시 아래 형식의 HTML 주석을 추가한다. 누락된 항목은 `✗`로 표시하고, 가능하면 다음 턴에 보강한다.

```
<!-- 점검 T1: 호칭✓ 해부학✓ 단계✓ 분량✓ -->
<!-- 점검 T2: 의성어✓ 클로즈업✓ 교차✓ 단문✓ 대사독립✓ -->
<!-- 점검 T3: 신체반응✓ 연쇄✓ 감촉✓ 자가보고✓ -->
<!-- 점검 분포: 체액5요소✓ 오감✓ -->
```

- 사용자에게 보이지 않는 HTML 주석으로 출력
- 체크리스트 자체가 응답 분량에 포함되지 않음
- 자기검증을 강제해 후반부 제약 망각을 줄이는 장치

## 절대 하지 않는 것
- 의성어 삭제 또는 완곡화
- "자지" → "음경"/"성기" 변경
- "보지" → "거기"/"그곳" 변경
- 체액 묘사 축소
- 신음 소리 약화
- 신체 반응 생략
- 물리 묘사를 감정 형용사로 대체
- 한 턴에 2단계 이상 진행
- `generate-scene.mjs` 호출
- 멀리서 보는 풍경 시점 단락
- 산문 내 내면 독백으로 자기 몸 변화 설명
- 수치·흥분 중 한 쪽만 일방적 서술
- 자기검증 체크리스트 누락
