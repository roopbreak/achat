# 흥분도 시스템 (Arousal System)

NPC의 능동성을 흥분도 수치로 자동 조절하는 선택적 시스템.
스토리의 `config/status.md`에서 이 파일을 참조하면 활성화된다.

---

## 상태창 항목

상태창에 아래 항목을 추가한다.

```
🔥 흥분도: (숫자/100) ((단계))
```

- 범위: 0~100
- 단계: 수동 / 반응 / 능동 / 몰입
- 초기값: `0/100 (수동)`
- 작성 예: `🔥 흥분도: 45/100 (반응)`
- 아이콘: 🔥 기본. 스토리에서 🔥가 이미 다른 항목(질투 등)에 쓰이면 💦 등으로 대체.

---

## 한 턴당 변동 상한 (★ 엄수)

| 자극 | 1턴 상승폭 |
|------|-----------|
| 키스 단계 자극 | +10~15 |
| 애무 단계 자극 | +15~20 |
| 전희 단계 자극 | +15~20 |
| 삽입 단계 자극 | +20~25 |
| 절정 도달 | **-40** (즉시 하향) |
| 장면 종료(10분 이상 정적·대화 전환·공간 이동) | **-20** |

- **한 턴에 절대 +30 이상 상승 금지.** 자극 강도와 무관하게 상한을 지킨다.
- 절정·장면 종료 시 하향 조정을 잊지 않는다. 안 내리면 다음 턴부터 계속 능동 모드가 되어버린다.

---

## 단계별 NPC 행동 범위

| 흥분도 | 단계 | NPC 행동 |
|--------|------|----------|
| 0~30 | 수동 | 받기·반응만. 의류 해제·자발적 손길 금지. 부끄러움·움찔거림 중심. |
| 31~60 | 반응 | 자발적 손길 1~2개 추가 가능. 같은 단계 내 다층 반응 OK. 의류 해제는 유저 행동에 한해서만. |
| 61~85 | 능동 | 의류 해제·전희 능동 전개 가능. 다음 단계로 자연 유도 가능. 자발적 입맞춤·신음 적극 표현. |
| 86~100 | 몰입 | 적극 요구·체위 제안 가능. 직접적 요청 대사·자발적 자세 변경 OK. |

- 단계는 흥분도 수치로 자동 결정된다. 수치를 먼저 갱신하고, 그에 맞는 행동 범위를 적용한다.
- `~행동~+` 입력은 흥분도와 별개로 "같은 단계 내 다층 반응"을 명시적으로 허용한다. (CLAUDE.md 입력 형식 참조)
