# 소설 산문 스타일 가이드

## 대사 형식 — 반드시 독립 줄

```
❌ 그녀가 말했다. "천천히 믿어줘도 돼." 목소리가 떨렸다.

✅
그녀가 한 발 물러섰다.

"천천히 믿어줘도 돼."

목소리가 가늘게 떨렸다.
```

- 대사 앞뒤에 빈 줄 삽입하여 독립 문단으로 분리
- 화자는 문맥과 문장 구조로 드러냄

## 금지 형식

- `**캐릭터명 |** "대사"` 레이블
- `*이탤릭 행동 묘사*`
- 턴 구분선 `---`
- `**[나]**` / `**[스토리]**` 레이블

## 문단 나눔 기준

다음 단위마다 빈 줄(`\n\n`)로 분리:
- 대사 (항상 독립)
- 장면·시점 전환
- 행동 → 내면 전환
- 감정 흐름 변화
- 시간 경과

## 문장 길이 — 엄수 사항

### 규칙
- 단문(주어+서술어만, 10자 이하)이 연속 2개 이상 나오면 **안 된다**
- 반드시 관계절·분사구·접속 표현으로 통합
- 단문은 감정 절정/반전 강조 시 문단당 **1개만** 허용
- 목표: 서술 문장 1개당 평균 40자 이상

### 예시

```
❌ 그녀가 돌아섰다. 눈이 빨개져 있었다. 입술이 떨렸다.
   (단문 3개 연속 — 절대 금지)

✅ 그녀가 천천히 돌아섰을 때, 눈가는 이미 빨갛게 물들어 있었고
   입술은 미세하게 떨리고 있었다.
```

```
❌ 발소리가 들렸다. 그가 왔다. 문이 열렸다.

✅ 발소리가 점점 가까워지더니 이윽고 문이 열리며
   그의 모습이 시야에 들어왔다.
```

## 행동 묘사

이탤릭 괄호 형식을 쓰지 않는다. 행동과 감정을 서술 문장 안에 녹인다.

```
❌ *그녀는 천천히 손을 거두며 무릎 위에 올려놓았다.*
✅ 그녀는 천천히 손을 거두어 무릎 위에 올려놓았다.
```

## 감정 묘사 기법

### 직접 서술 대신 행동으로
```
❌ 그는 슬펐다.
✅ 그의 시선이 창밖으로 흘렀고, 커피잔을 감싼 손가락 끝이
   미세하게 떨리고 있었다.
```

### 감각 활용
- 시각: 빛, 색감, 시선의 방향
- 청각: 숨소리, 침묵의 무게, 멀리서 들리는 소리
- 촉각: 온도, 질감, 접촉의 강도
- 후각: 향기, 익숙한/낯선 냄새
- 미각: 맛의 변화, 입안의 건조함

### 내면과 외면의 교차
한 문단 안에서 외면 묘사와 내면 독백을 자연스럽게 교차:
```
그녀의 손이 테이블 위를 더듬어 물잔을 찾았다. 목이 말랐다.
정확히는 목이 마른 게 아니라 무언가를 삼켜야 했다 —
지금 터져 나오려는 말을, 아직은 꺼내면 안 되는 말을.
```

## 복장 묘사

### 원칙
- 복장은 캐릭터 성격·현재 감정·상황을 반영하는 서술 장치다
- 같은 옷을 계속 입히지 않는다. 장면 전환(장소 이동, 날짜 변경, 외출/귀가)이 있으면 복장 변화를 반영
- 복장 나열이 아니라 행동·시선·감각 묘사 안에 녹인다

### 예시

```
❌ 그녀는 흰색 블라우스에 검은 슬랙스를 입고 있었다. (직접 나열)

✅ 블라우스 소매를 한 번 접어 올린 손목이 LED 조명 아래서 유난히 하얗게 보였다.
✅ 바람이 카디건 자락을 들추자 그녀가 무심하게 여몄다.
✅ 넥타이를 풀어 주머니에 구겨 넣은 채 돌아온 그의 얼굴에는 오후 내내의 피로가 묻어 있었다.
```

### 복장이 감정을 반영하는 경우
```
평소 단정한 캐릭터가 대충 입고 나옴 → 내면이 무너졌다는 신호
긴장한 캐릭터가 평소보다 차려입음 → 무장하듯 감정을 감추는 것
편안한 차림으로 변화 → 상대에 대한 경계가 풀리고 있다는 신호
```

### 복장 변경 트리거
- 날짜가 바뀌면 반드시 복장 변경
- 장소가 바뀌면 상황에 맞게 복장 고려 (집→외출, 병원→퇴근 등)
- 감정의 큰 변화가 있을 때 복장 디테일로 간접 표현

## 장면 전환

### 시간 경과
```
직접: "세 시간이 흘렀다."보다
간접: "찻잔의 온기가 식어 손끝이 차가워질 무렵,"
```

### 공간 이동
```
직접: "거실로 갔다."보다
간접: "현관을 지나 거실의 불빛이 시야에 들어왔을 때,"
```
