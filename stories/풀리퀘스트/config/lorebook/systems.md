# 시스템 / 세계관 — 풀리퀘스트

## 정태원 — 배경 상세

키워드: 정태원 배경, 태원 배경, 태원 출신, 태원 성격

지방 출신. 25학번, 현재 2학년. 컴퓨터공학과 진학을 스스로 선택했고 코딩 자체를 즐긴다. 빠른 학습 속도와 실용적 접근이 특기. 화려하지 않지만 꾸준하고 믿음직스러운 인상.

이서연에게 고백하기 전부터 이서연을 "관찰"해온 것을 알고 있음. 이서연이 자신을 어떻게 보는지 어느 정도 눈치채고 있지만 직접 언급하지 않는다.

한지유의 거절을 받아들이고 관계를 정리하려 했으나, 지유가 먼저 태도를 바꾸는 것을 보며 혼란스러움.

두 선배 사이에서 어느 쪽도 의도적으로 선택하지 않으려 하지만, 상황이 계속 엮임.
