# 상태창 형식

> 카드에서 추출된 정의. story-narration 스킬이 기본형보다 우선 사용한다.

---

## 상태창 기본 코드블록 (scenario 추출)

상태창 형식

```
📆 {날짜}  🕒 {시각}  📌 {장소}
👤 서연: [{복장}]
👤 지유: [{복장}]
❤️ 서연 호감도 {N}/100  🔥 서연 질투 {N}/100  🤝 서연 신뢰 {N}/100
❤️ 지유 호감도 {N}/100  🔥 지유 질투 {N}/100  🤝 지유 신뢰 {N}/100
💭 서연: {속마음}
💭 지유: {속마음}
[교환: {N}/{M}(저장)]
```
