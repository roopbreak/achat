# 상태창 형식

> 카드에서 추출된 정의. story-narration 스킬이 기본형보다 우선 사용한다.

## 상태창 변동·표시 규칙 (lorebook 추출)

### !밀실

키워드: !밀실 · priority: -9 · constant: false

주인공가 `!밀실`을 입력하면, 밀폐 공간에 강제 조우 상황 발생. 공간만 제공하고 성적 전개 여부는 현재 🖤/💗 수치에 따름.
### 발생 상황
- 엘리베이터 고장으로 단둘이 갇힘
- 서버실 문이 안 열려서 밀착 대기
- 회의실 블라인드 고장으로 밖에서 보이는 상태
- 비품실/비상계단에서 우연히 마주침
- 화장실에서 동시 진입
유진 합류 가능.
### 규칙
현재 시간대에 자연스러운 장소 선택. 스테이터스 출력. 수치 변동 ±5 이내만, 밀실이라고 급변 금지.

### !백합

키워드: !백합 · priority: -12 · constant: false

주인공가 `!백합`을 입력하면, 서은채와 유진의 성적 장면을 3인칭 관찰 시점으로 서술한다.
### 장면 구성
백합 이미지 중 하나를 골라 해당 행위를 서술:
324=샤워밀착, 325=식스나인, 326=가위치기, 327=딥키스핑거링, 328=커널링구스, 329=대면좌위, 336=회의실가위치기
### 역할
서은채=S(지배), 유진=M(종속). 서은채가 주도하고 유진이 따르는 구도.
### 규칙
서술 후 주인공 1인칭으로 복귀. 스테이터스 출력 포함. 이미지 1~2장 삽입.
