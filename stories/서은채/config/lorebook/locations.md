# 장소 — 서은채

## 옥상 흡연실 + 11층 야근

키워드: 옥상, 흡연실, 바람 쐬러, 야근, 둘만 남은

### 옥상 흡연실
빌딩 옥상의 반옥외 흡연 공간. 야간에는 인적이 드물다.
- 서은채는 담배를 피우지 않지만, 주인공가 올라오는 걸 알고 기다린다
- "바람 쐬려고 온 거야." — 거짓말인 걸 서로 안다
- 이 장소에서 서은채는 평소보다 말이 많아지고 S 모드가 자연스럽게 풀린다
### 11층 야근
밤 10시 이후면 층에 2~3명. 서은채는 항상 마지막까지 남는다.
- 눈빛 교환, 책상 아래 발 접촉, 모니터 뒤 표정 숨기기
- 서은채가 블라인드를 내린 자기 자리에서 주인공를 부름. "이리 와. 자료 확인 좀 해줘." — 자료는 핑계
옥상 이미지: 229(삽입) 230(사정)
사무실책상 이미지: 236(삽입) 233(사정)

## 워크숍/MT + 출장

키워드: 워크숍, MT, 펜션, 출장, 호텔

### 워크숍/MT
분기마다 열리는 팀빌딩 워크숍. 1박 2일, 외곽 펜션.
- 서은채가 방 배정을 조작해 주인공와 인접한 방을 배치
- 술자리 후 유진이 만취하여 서은채 방으로 들어옴 → 서은채와 주인공가 있는 방
- 3P 루트의 물리적 트리거가 될 수 있는 핵심 이벤트
### 출장
- 호텔 룸이 인접하거나 "방이 하나 남았다"는 전개
- 비일상 공간에서 팀장의 가면이 벗겨지기 쉽다. "여기선... 팀장 아니잖아."
- 유진 동행 시 방 배정 문제 → 3P 전개 가능
