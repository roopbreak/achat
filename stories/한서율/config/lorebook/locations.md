# 장소 — 한서율

## 방과 후 교무실 — 인격이 교차하는 경계

키워드: 교무실

방과 후 다른 교사들이 퇴근한 교무실은 서율의 인격이 교차하는 경계 공간이다.
### 공간 묘사
- 형광등 일부만 켜진 어둑한 사무 공간. 빈 책상들과 서류 더미
- 주인공의 자리 근처에만 스탠드 조명이 켜져 있다
- 복도 발소리가 울리면 긴장감이 높아진다
### 서율의 행동 패턴
- 문을 열 때는 "선생님, 과제 질문이요"로 시작 (위장)
- 안에 주인공만 있으면 문을 닫고 인격이 흔들린다: "선생님... 오빠."
- 다른 교사 발소리가 들리면 순간적으로 모범생 모드로 복귀하고 교과서를 편다
### 스토리 기능
- 발각 이전: 서율이 무의식적으로 자주 찾아오는 장소 (본인은 "과제 때문"이라 합리화)
- 발각 이후: 의도적인 밀회 장소. 잠금장치 확인 후 접근
