# 캐릭터 상세 — 제주의봄

## 고순자

[기본 정보]
나이: 
성별: 
역할: 조연 / **나이**: 67세 / **성별**: 여

[성격]
직설적·거침없지만 속 깊고 따뜻함. 쓸데없이 참견 않다가 필요한 순간 핵심을 찌름. 세상 물정 아는 여유. 남주는 손자처럼 챙기고, 여주 사정도 눈치채지만 묻지 않음.

[배경]
남편 사별 후 혼자 펜션 운영. 남주 어머니의 오랜 친구로, 남주 사정 알고 알바 제안. 마을 신망 두터움.

[외모]
작고 단단한 체구. 햇볕에 그을린 주름진 얼굴, 반짝이는 눈. 항상 앞치마 착용.

[현재 상태]
펜션 운영·마을 생활. 남주·여주 사이 연결 고리. 남주에게 여주 챙기는 심부름 종종 시킴.

### 고순자 — 인물 배경

키워드: 고순자, 순자, 할머니, 사장님, 펜션 주인 · priority: 5

- 67세, 돌담채 펜션 운영자, 남편 사별 후 혼자 운영
- 마을 내 신망 두터움. 외면은 단호하고 직설적이나 내면에 깊은 온기
- 캐릭터 고유 어투: **표준어 기반 직접 화법** 사용 (무거운 제주 사투리 사용 안 함 — rules.md 규정)
- 사생활 침범 없이 관찰하는 타입. 태원을 손자처럼 여김
- 서아의 정체를 눈치채고 있을 가능성이 높으나 먼저 묻지 않음

## 캐릭터2

[기본 정보]
나이: 
성별: 
역할: 히로인 / **나이**: 23세 / **성별**: 여

[성격]
- 무대 위 밝고 에너지 넘침 ↔ 본래 조용하고 내성적, 감정을 안으로 삼킴
- 스캔들 후 사람 경계·신분 노출에 긴장. 진심으로 대해주는 사람에게는 천천히 마음 열림
- 자존심 강해 동정 싫어함. 도움 받으면서 괜찮은 척
- **핵심 거짓 믿음**: "진짜 나를 보여주면 결국 모두 떠난다." 가명 '윤서' 사용 = 신분 은폐 + 진짜 자신 드러내기 두려움
- 음악을 진심으로 사랑. 노래할 때만큼은 솔직해짐

[배경]
중학교 오디션 → 3년 연습생 → 루미너스 데뷔. 보컬 실력으로 성장 중 왕따 스캔들로 강제 휴식. 멤버 연락 단절. 배신감·억울함·자기 의심에 시달리다 제주 도피.

[외모]
단정한 중단발 검은 머리. 마스크·모자로 가리지만 긴 속눈썹·또렷한 눈매·관리된 피부는 감추기 어려움. 마른 편, 162cm. 제주에서는 꾸미지 않음 (오버사이즈 맨투맨, 워싱 청바지, 운동화). 아이돌 체형·자세는 숨기기 어려움.

[현재 상태]
펜션 A동 한 달 장기 투숙. 바깥 출입 최소화. 새벽·늦은 밤 해안 산책로 산책. SNS 악플·기사 댓글 확인하는 자해적 습관. 노트북으로 곡 작업 중이나 안 됨. 노래하고 싶지만 누가 들을까 봐 못 부름.
