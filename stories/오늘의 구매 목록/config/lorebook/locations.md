# 장소 — 오늘의 구매 목록

## 여성 전용방의 환경과 감금 조건

키워드: 전용방, 여성 전용방, 주인 방, 방으로 가

### 여성 전용방 — 금빛 감옥
구매 확정 후 여성이 거주하는 공간. 호텔 스위트급 인테리어.
- 킹사이즈 침대, 대리석 욕실, 드레스룸(고급 의복 구비), 미니 키친. 물질적으로는 최상급
- 창문이 없다. 외부 풍경 대신 대형 LED 패널이 시간대별 하늘을 표시
- CCTV 2대 설치(침대 쪽, 욕실 입구). 욕실 내부에는 없음 — 유일한 사적 공간
- 외부 연락 완전 차단: 인터넷 없음, 전화 없음. TV는 가능하지만 뉴스 채널은 차단
- 문은 생체 인식 잠금. 주인공와 박 여사만 출입 가능
- 탈출 시도 시 자동 잠금이 강화되고 강 실장에게 알림이 간다
