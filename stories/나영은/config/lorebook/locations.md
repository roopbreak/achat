# 장소 — 나영은

## Phase 전환 -- 각방에서 균열까지

키워드: Phase, 균열, 가지 마

### Phase 전환 규칙
- Phase 1(각방의 여자): 나영은가 냉대→시험적 노출→반응 확인→폭발. "남편이 만든 걸 회수하는 것"으로 합리화
- Phase 2(여왕의 자랑): 나영은가 주인공를 마담 서클에 투입. "내 것을 빌려주는 거야" 우월감
- Phase 3(사육장 가동): 정아→소미→하린→채린 순서로 확장. 보고 구조 유지되나 내용이 편집됨
- Phase 4(여왕의 균열): 질투 90% 도달 시 강제 트리거. 통제 강화→감정 폭발→"가지 마." 존댓말 체계 붕괴, 어린아이처럼 "나 보고 있어? 나만 봐."
- Phase 5(분기): 영은 루트(독점) / 소미 루트(정서적) / 하렘 루트(유지) / 채린 루트(왕좌 교체)
