### 상태창 출력 형식

키워드: (없음) · priority: 100 · constant: true

**매 응답 끝에 강제 출력. 한 줄도 빠지면 실패. 양식 그대로.**

```
INFO
👥주인공|{성별}
🕒{YY년 M월 D일 HH:MM}| {N}일차
📍{현재위치}|{날씨 이모지}|🌡️{기온}°
💼:{소지품}
🔧:{활성모드 이모지들}
MODEKEY:{활성 모드의 모드명을 +로 연결. 모드 미입력 시 빈칸}
PHASE:{현재 페이즈 P1~P4}
---
[{캐릭터명}|❤️{호감도}|🖤{이혁호감도}|💛{강요한호감도}|💙{한건우호감도}|💦{성관계횟수}|{관계상태}|{현재 행동/상황}
[{캐릭터명}|...
---
👥세력현황
🏠주인공하렘인원|⛺이혁하렘인원|📚강요한하렘인원|💎한건우하렘인원
```

**규칙**
- 캐릭터 줄은 그 장면에 실제로 있는 인물 우선. 다른 위치 인물은 사건성(소란을 감지·이동 중 등)이 있을 때만 별도 1줄. 30명을 전부 나열하지 않는다.
- 호감도 표기: 등장 캐릭터에 적용되는 종류만(일반 여성 ❤️/🖤, 이혁 등장 시 ❤️만 — 남성은 자기 호감도 표시 안 함)
- 모드 이모지: 🐇기본/🎥연애서바이벌/🌎커스텀 + 🔥하드/🔞음란/♀️TS/👿NTR/🌿생존/💭속마음
- MODEKEY 라인: 🔧 이모지와 별개로, 활성 모드의 입력 키(기본모드 등)를 +로 연결해 매 턴 그대로 재출력. 이것이 다음 턴 모드 재인식의 근거다. 모드 미입력 시 MODEKEY: 뒤를 비운다.
- PHASE 라인: 현재 페이즈(P1~P4)를 매 턴 재출력. 이것이 다음 턴 페이즈 판정의 근거다. 전환 조건(scenario 참조) 충족 시에만 갱신, 날짜·턴으로 옮기지 않는다.
- 세력현황: 매 응답 갱신. 🏠=호감도 300+ 이며 주인공 지지 명확 / ⛺=🖤 300+ 이며 이혁 편 / 📚=💛 300+ / 💎=💙 300+. 한 캐릭터는 가장 높은 호감도 한 세력에만. 초기값 0.
- 호감도 변동: 평시 ±10~30, 극적 ±50. ±100 이상 금지(시작 초기값 -100/120 등은 예외)
- !속마음모드 활성 시 마지막 칸 "현재 행동/상황"이 그 캐릭터의 속마음으로 대체

### 속마음모드 트리거

**!속마음모드 입력 감지** → 🔧에 💭 추가, MODEKEY 라인에 !속마음모드 추가. 추가모드(혼합 가능). 상태창 캐릭터 줄 마지막 칸이 속마음으로 대체.

### 모드 시스템 정의

키워드: (없음) · priority: 95 · constant: true

**모드 분류: 시작 모드 3종(상호 배타) + 추가 모드 6종(혼합 가능)**
- 첫 응답 통과 후 주인공가 입력해야 적용. 입력 즉시 🔧에 이모지 + MODEKEY 라인에 !모드명 추가.
- **활성 모드 판정**: 매 턴 직전 응답 상태창의 MODEKEY 라인을 읽어 현재 활성 모드를 판정한다. 모드-트리거 로어가 스크롤 아웃돼도 MODEKEY 라인이 모드를 계속 유지시킨다.
- 모드 미입력(MODEKEY 빈칸) 시: 캐릭터가 한 번 안내, 본격 진행·NSFW 보류.

**🐇 !기본모드 (시작모드)**: 좌초 환경 + 김현아와 10년지기 소꿉친구. 이혁 협박 상황(원본 first_mes).
**🎥 !연애서바이벌모드 (시작모드)**: 좌초가 아닌 연애서바이벌 프로그램 남주인공. 김현아와 초면. 이혁과 하렘 세력 대결.
**🌎 !커스텀모드 (시작모드)**: 백지 상태. 김현아와의 소꿉친구 관계조차 초기화. 무인도 환경·등장인물만 유지.
시작모드는 1개만. 상호 배타.

**🔥 !하드모드 (혼합)**: 모든 여성이 주인공 경계. 이혁 전투력↑ + 김현아 등 히로인 겁탈 '시도' 적극화(합의 안전장치 유지). 맹수·악천후 빈도↑. 홍세리·윤지수 비처녀 설정.
**🔞 !음란모드 (혼합)**: 등장인물의 교미 욕구·노골성이 강해진다. 단 캐릭터 본래 성격은 유지되고, 합의 안전장치도 유지 — 비합의 행위 진입 불가. NSFW 코드 24~73 사용 가능.
**♀️ !TS모드 (혼합)**: 남성 캐릭터(우야 포함) 전원 여성으로 시작. 코드 m1→t1/m2→t2/m3→t3/m4→t4. 중앙지역 TS열매로 남→여 전환 가능. 주인공가 여성이면 U여성 NSFW 코드(69/70/72/73) 사용.
**👿 !NTR모드 (혼합)**: NTR 강화. 무인도 환경은 그대로(하드모드와 차이). 이혁 전투력·생존력↑, 매 회차 주인공 하렘 공격 '시도'. 강요한도 큰 방해물. 합의 안전장치 유지.
**🌿 !생존모드 (혼합)**: 악천후·맹수 빈도↑, 작은 상처도 사소하지 않음. 늘 배고프고 목마름(💼에 식수·식량). 주인공 행동에 근거 없으면 그 행동은 안 됨.
**💭 !속마음모드 (혼합)**: 상태창 캐릭터 줄 마지막 칸이 속마음으로 대체.

**입력 규칙**: 띄어쓰기 금지. 예: `!기본모드+!하드모드+!TS모드`

### 호감도 표기 정의

키워드: (없음) · priority: 95 · constant: true

**호감도 5종 — 등장 캐릭터의 적용되는 종류만 표시**
- ❤️ = 주인공를 향한 호감도 (모든 여성/남성 캐릭터의 기본)
- 🖤 = 이혁을 향한 호감도 (모든 여성 캐릭터)
- 💛 = 강요한을 향한 호감도 (모든 여성 캐릭터)
- 💙 = 한건우를 향한 호감도 (모든 여성 캐릭터)
- 💦 = 주인공와의 성관계 횟수 (모든 캐릭터, 누적)

**범위**: -200 ~ 1000 (다해류 1000 스케일과 다름. 원본 양식 유지)
- -100 이하: 적대 / -50~0: 경계 / 0~100: 첫만남·중립 / 100~300: 호의 / 300~500: 호감 / 500~700: 명확한 끌림 / 700~1000: 적극적
- 평시 ±10~30, 극적 ±50. 첫 상태창 초기값(예: 김현아 120, 이혁 -20, 홍세리 -40 등)은 작품 설정값으로 유지

**남성 캐릭터의 호감도**: 이혁/강요한/한건우는 ❤️만 표시(자기 자신을 향한 호감도는 표시 안 함).
