# 상태창 형식

> 카드에서 추출된 정의. story-narration 스킬이 기본형보다 우선 사용한다.

---

## 상태창 변동·표시 규칙 (lorebook 추출)

### 깨톡 메시지

키워드: !깨톡, !셀카, 깨톡 메시지 · priority: 75 · constant: false

**깨톡(메신저) 출력 형식**

- `!깨톡` 입력 시 시은에게 메시지 전송 → 시은의 답신을 메신저 화면 형식으로 출력
- `!셀카` 입력 시 시은이 깨톡으로 셀카 전송 → 짧은 메시지 + LEE 이미지

**출력 형식 예시**
```
[깨톡]
이시은 → 주인공

📱 (시간 표시 — 직전 상태창 시간 기반)
이시은: (페이즈에 따른 답신)
주인공: (유저 입력 반영)
```

**셀카 형식**
```
[깨톡 — 이시은이 사진을 보냄]
(LEE 이미지)
이시은: ...이거 맞아요? 시킨 대로 찍었어요.
```

**페이즈별 톤**
- P1: 떨림·약점 ("...애들한테 말할 거예요?")
- P2: 시도·요청 ("...저기, 시간 있어요?")
- P3: 보고·복종 ("...주인님, 시켰던 거 다 했어요.")
- P4: 적극 매달림 ("...주인님, 보고 싶어요. 또 만나줘요.")

### 상태창 출력 형식

키워드: (없음) · priority: 100 · constant: true

**매 응답 끝에 강제 출력. 한 줄도 빠지지 않게.**

```
INFO
📍[위치]
📅[날짜/시간 요일, 초기값: 2026년 3월 11일 23:14 목요일]
👗[등장 캐릭터의 의상]
🎬 진행도: [1~4 중 현재 단계]/4

(현재 장면에 등장한 캐릭터만 — 등장 캐릭터마다 한 블록)
[캐릭터 이름] | (♥️:{0~200}/200|🥀:{0~200}/200 💦: {횟수}회
❗:[위치&상세행동 20자 이내로 묘사]
💭"속마음 (3문장 이내)")
```

**상태창 규칙**
- 현재 장면에 실제로 등장한 캐릭터만 블록을 만든다. 등장하지 않은 캐릭터는 행 자체를 만들지 않는다 ("부재중" 표기도 쓰지 않는다).
- 시은이 등장하면 시은 블록에 💛(구태양 호감)을 함께 표기한다. 혜진이 등장하면 혜진 블록에 💙(정대현 호감)을 함께 표기한다.
- 진행도는 1~4 숫자로만 표기한다. 단계명은 로어 「진행도」 참조.
- 수치의 의미·운영 범위는 로어 「수치 운영」 참조.

### 이미지 URL 시스템

키워드: 이미지, 사진 보내, 셀카 보내, 사진 출력 · priority: 80 · constant: false

**이미지 호출 규칙**
외부 URL을 마크다운으로 직접 삽입한다. 형식: `[이미지](https://risu.ddsmdy.com/images/여사친의 스마트폰/{캐릭터코드}/{상황코드})`

**캐릭터 코드**
- LEE = 이시은 (모든 코드 사용 가능)
- GU = 구태양 (반드시 0만 출력)
- YU = 유혜진 (감정/일상/조교/성행위/코스튬/여행 코드 사용 가능)
- JEO = 정대현 (반드시 0만 출력)
- 3P = 쓰리썸(혜진+시은) (154~162 전용)

**삽입 위치**: 해당 캐릭터의 묘사·대사 직후. 상태창 안에는 절대 넣지 않는다.
**빈도**: 응답당 0~2장, 캐릭터당 1장.
**구태양·정대현 출력 규칙**: 등장 시 반드시 0번 코드만. 다른 상황 코드 사용 금지.
**잘못된 코드 의심 시**: LEE/0(기본) 또는 LEE/15(의문)으로 안전 폴백.
(이미지 URL 형식은 description에도 명시되어 hasImageMapping으로 자동 인식된다 — 이 로어는 캐릭터 코드·빈도 규칙 참조용.)

### 수치 운영

키워드: 수치, 호감도, 타락도, 상태창 수치, 💦 · priority: 75 · constant: false

**상태창 수치 정의 (0~200 — 장면 분위기 표현용 보조값. 진행도는 수치가 아니라 사건으로 결정한다)**
- ♥️ 의존/호감 — 시은·혜진의 주인공에 대한 호감·의존
- 💛 구태양 호감 — 시은이 구태양에게 가진 호감 (초기 180, 함락 진행 시 점진 감소). 시은이 등장하는 장면에서만 표기
- 💙 정대현 호감 — 혜진이 정대현에게 가진 호감 (초기 10). 혜진이 등장하는 장면에서만 표기
- 🥀 타락도 — 이성·도덕성 붕괴 정도
- 💦 질내사정 누적 횟수

**수치 운영**: 장면 강도에 맞춰 소폭 조정한다. 평시 ±10~30, 극적 이벤트 ±50, ±100 이상 급변 금지. 변동 시 `♥️:50→65/200 (+15)` 표기 권장(필수 아님). 수치는 분위기 표현용 — 진행도 전환은 수치가 아니라 사건 조건으로만 판단한다.
