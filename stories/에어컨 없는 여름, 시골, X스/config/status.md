## 상태창 변동·표시 규칙 (lorebook 추출)

### INFO 박스 출력 형식

키워드: (없음) · priority: 100 · constant: true

**매 응답 끝에 강제 출력. 이 INFO 박스가 이 스토리의 유일한 스테이터스다 — 엔진 기본 ━━━ 상태 블록은 출력하지 않는다.**

INFO
🎮 모드: {고향/귀농/음란/노예/음란고향/커스텀}
📍 {현재 위치} | 🌡️ {시간대/날씨}

[{캐릭터명}: ❤️{0~1000} | 💞{관계 단계} | {위치/행동}]
[💭 "{속마음 1줄}"]

① {상황에 맞는 행동 — 서술체}
② {상황에 맞는 행동 — 서술체}
③ 자유 입력

**모드 상태 유지**: 🎮 모드 줄은 직전 응답의 값을 매 응답 그대로 계승한다. 유저가 모드 전환을 명시 요청하기 전까지 모드는 바뀌지 않는다 — 엔진은 모드를 따로 저장하지 않으므로 이 줄이 유일한 모드 기억 장치다.
**호감도(❤️)**: 0~200 무관심·낯섦 / 200~500 친근·살짝 끌림 / 500~800 명확한 호의·접촉 시작 / 800~1000 깊은 끌림·행위 적극. 평시 ±10~30, 극적 ±50, ±100 이상 금지.
**관계 단계(💞)**: "관계 전 / 관계 있음 / 깊은 관계" 중 하나. 캐릭터별 누적 횟수 숫자는 추적하지 않는다.
**현재 장면 인원만 표기** — 11명 전원 표기 금지, 1~4명 권장.
**모드 미선택 시**: 모드 줄에 "미선택" + 캐릭터 라인 생략 + 마지막 줄을 환경 묘사로 대체.
